## 📋 四、图例说明

所有图中使用统一的配色方案：

- 🟡 **单质节点**：黄色底框（如 Na、Fe、Cl₂）
- 🔵 **普通化合物节点**：蓝色底框
- 🔴 **高频考点节点**：红色底框，含「🔴」标记
- ➡️ **普通箭头**：黑色，标注反应物和条件
- ➡️ **高频考点箭头**：红色加粗
- 🔵 **酸性方向**：蓝色箭头
- 🟢 **碱性方向**：绿色箭头
- ⬜ **考点总结区**：底部白色框，列出该元素近十年高频考点

## 🔄 五、推荐学习路径

1. **先看主族比较图** → 建立"一族元素有共性也有个性"的框架意识
2. **逐个元素深入** → 按 Na→Fe→Al→Cu→Cl→S→N→C→Si 顺序
3. **对比相邻元素** → C vs Si、N vs P、O vs S、Na vs K
4. **综合推断训练** → 配合各册 `substance_transformation*.md` 中的推断题组
