## 第一章 物质及其变化

> **地位**：全册方法论的根基，氧化还原贯穿后续所有元素化合物学习。

![第一章 物质及其变化](./knowledge_map_ch1.svg)

### 第一节 物质的分类及转化

```
物质分类树
├── 混合物（空气、溶液、合金）
└── 纯净物
    ├── 单质
    │   ├── 金属单质（Na, Fe, Al）
    │   ├── 非金属单质（O₂, Cl₂, C）
    │   └── 稀有气体
    └── 化合物
        ├── 酸（HCl, H₂SO₄, HNO₃）
        ├── 碱（NaOH, Ca(OH)₂）
        ├── 盐（NaCl, Na₂CO₃）
        └── 氧化物
            ├── 酸性氧化物（CO₂, SO₂）
            ├── 碱性氧化物（Na₂O, CaO）
            └── 两性氧化物（Al₂O₃）
```

| 概念 | 含义 | 示例 |
|------|------|------|
| **同素异形体** | 同种元素组成的不同单质 | O₂/O₃，金刚石/石墨/C₆₀ |
| **分散系** | 一种物质分散到另一种物质中 | 溶液、胶体、浊液 |
| **胶体** | 分散质粒子直径 1~100 nm | Fe(OH)₃胶体、AgI胶体 |
| **丁达尔效应** | 光通过胶体产生光路 | 鉴别胶体与溶液 |

### 第二节 离子反应

**电解质与非电解质**
- **电解质**：水溶液或熔融状态下能导电的化合物（酸、碱、盐、金属氧化物）
- **非电解质**：两者都不能导电的化合物（蔗糖、酒精、CO₂）
- **强电解质**：完全电离 → HCl、H₂SO₄、NaOH、NaCl
- **弱电解质**：部分电离 → CH₃COOH、NH₃·H₂O、H₂O

**离子反应发生的条件**（满足其一即可）：
1. 生成**沉淀**（如 AgCl↓、BaSO₄↓）
2. 生成**气体**（如 CO₂↑、H₂↑）
3. 生成**弱电解质**（如 H₂O、CH₃COOH）

**离子方程式书写——四步法**：
```
写 → 拆 → 删 → 查
```
- **可拆**：强酸、强碱、可溶性盐（拆成离子）
- **不可拆**：沉淀、气体、弱电解质、氧化物、单质

> ⚠️ **常见错误提醒**：CaCO₃ 是沉淀不能拆！CH₃COOH 是弱酸不能拆！

### 第三节 氧化还原反应

**核心口诀**：**"升失氧还，降得还氧"**

| 特征 | 化合价 | 电子 | 反应类型 | 角色 |
|------|--------|------|----------|------|
| 升 | 升高 | 失去 | 被氧化 | 还原剂 |
| 降 | 降低 | 得到 | 被还原 | 氧化剂 |

- **还原剂** = 化合价升高 = 失电子 = 被氧化 = 具有还原性
- **氧化剂** = 化合价降低 = 得电子 = 被还原 = 具有氧化性

**常见氧化剂**：O₂、Cl₂、KMnO₄、HNO₃、浓H₂SO₄、Fe³⁺
**常见还原剂**：Na、Al、Zn、C、CO、Fe²⁺、KI

**四大基本反应 与 氧化还原的关系**：
- 化合反应：**部分**属于
- 分解反应：**部分**属于
- 置换反应：**全部**属于 ✓
- 复分解反应：**全部不**属于 ✗

**配平方法**：化合价升降法 → 得失电子守恒

## 第二章 海水中的重要元素——钠和氯

> **地位**：元素化合物学习的开端 + 物质的量（全册计算基石）。

![第二章 钠和氯](./knowledge_map_ch2.svg)

### 第一节 钠及其化合物

#### 钠单质 Na
- 银白色、质软、密度比水小
- 保存：**煤油中**（隔绝空气和水）

| 反应 | 方程式 | 现象 |
|------|--------|------|
| 常温与O₂ | 4Na + O₂ = 2Na₂O | 表面变暗 |
| 点燃与O₂ | 2Na + O₂ → Na₂O₂ | 黄色火焰，淡黄色固体 |
| 与水 | 2Na + 2H₂O = 2NaOH + H₂↑ | **浮、熔、游、响、红** |

#### 过氧化钠 Na₂O₂
- 淡黄色固体
- 2Na₂O₂ + 2H₂O = 4NaOH + O₂↑
- 2Na₂O₂ + 2CO₂ = 2Na₂CO₃ + O₂
- 用途：**供氧剂**（呼吸面具/潜水艇）、漂白剂

#### Na₂CO₃ vs NaHCO₃ 对比

| 性质 | Na₂CO₃（纯碱/苏打） | NaHCO₃（小苏打） |
|------|---------------------|-------------------|
| 外观 | 白色粉末 | 白色细小晶体 |
| 热稳定性 | 稳定，不分解 | 2NaHCO₃→Na₂CO₃+H₂O+CO₂↑ |
| 与酸反应 | 较慢，分步 | **更剧烈，一步完成** |
| 溶液碱性 | 较强 | 较弱 |
| 与CaCl₂ | CaCO₃↓（白色沉淀） | 不反应 |

**鉴别方法**：加热（是否分解）、加CaCl₂（是否沉淀）、加盐酸速率

#### 焰色试验
- 本质：**物理变化**（电子跃迁）
- Na → **黄色** | K → **紫色**（透过蓝色钴玻璃观察）

### 第二节 氯及其化合物

#### 氯气 Cl₂
- **黄绿色**气体，有毒，密度大于空气
- 实验室制法：MnO₂ + 4HCl(浓) --加热→ MnCl₂ + Cl₂↑ + 2H₂O

| 反应类型 | 方程式 | 现象/要点 |
|----------|--------|-----------|
| 与Na | 2Na + Cl₂ --点燃→ 2NaCl | 白烟 |
| 与Fe | 2Fe + 3Cl₂ --点燃→ 2FeCl₃ | 棕褐色烟 |
| 与H₂ | H₂ + Cl₂ --点燃→ 2HCl | 苍白色火焰 |
| 与水 | Cl₂ + H₂O ⇌ HCl + HClO | **氯水成分**：H⁺, Cl⁻, ClO⁻, HClO, Cl₂ |

#### HClO（次氯酸）
- 弱酸，具有**漂白性**（氧化漂白，不可逆）
- 光照分解：2HClO --光照→ 2HCl + O₂↑

#### 漂白粉
- 制取：2Cl₂ + 2Ca(OH)₂ = Ca(ClO)₂ + CaCl₂ + 2H₂O
- 有效成分：**Ca(ClO)₂**
- 漂白原理：Ca(ClO)₂ + CO₂ + H₂O = CaCO₃↓ + 2HClO

#### Cl⁻ 检验
AgNO₃溶液 + 稀HNO₃ → **白色沉淀**（AgCl），不溶于稀HNO₃

### 第三节 物质的量

> 🔑 **全册最重要的计算工具！**

**核心概念**
- 物质的量（n），单位：**mol**
- 阿伏伽德罗常数：**Nₐ ≈ 6.02×10²³ mol⁻¹**
- 摩尔质量（M）：单位 g/mol，数值 = 相对原子/分子质量
- 气体摩尔体积（Vₘ）：标况下 = **22.4 L/mol**

**四组换算关系**（以 n 为中心）：
```
m(质量,g) --÷M(摩尔质量)--> n(物质的量,mol) <--÷22.4-- V(气体体积,标况,L)
                                     ↑
                                ÷V(aq) × c
                                     ↓
                              c × V(aq) = n
```

**阿伏伽德罗定律**：同温同压下，V₁/V₂ = n₁/n₂

**一定物质的量浓度溶液的配制**：
```
步骤：计算 → 称量 → 溶解 → 冷却 → 转移
    → 洗涤 → 定容 → 摇匀 → 装瓶贴签
```
仪器：**容量瓶**、天平、烧杯、玻璃棒、胶头滴管

**误差分析**（核心：c = n/V）：
- 俯视定容 → V偏小 → **c偏高**
- 仰视定容 → V偏大 → **c偏低**

## 第三章 铁 金属材料

> **地位**：重点掌握铁三角和铝的两性。

![第三章 铁 金属材料](./knowledge_map_ch3.svg)

### 第一节 铁及其化合物

#### 铁单质 Fe
- 银白色金属，能被磁铁吸引
- 3Fe + 2O₂ --点燃→ Fe₃O₄（火星四射）
- Fe + 2H⁺ = Fe²⁺ + H₂↑（与非氧化性酸）
- 常温下浓H₂SO₄/浓HNO₃ → **钝化**

#### 铁的氧化物

| 氧化物 | FeO | Fe₂O₃ | Fe₃O₄ |
|--------|-----|--------|-------|
| 颜色 | 黑色粉末 | 红棕色 | 黑色 |
| 铁价态 | +2 | +3 | +2和+3 |
| 俗称 | — | 铁锈 | 磁铁矿 |

#### ★ Fe²⁺ 与 Fe³⁺ 的转化（铁三角）

```
       强氧化剂(Cl₂, HNO₃, H₂O₂)
Fe ─────────────────────────────────→ Fe³⁺
│                                       │
│ H⁺/Cu²⁺（弱氧化剂）                    │ Fe/Cu/KI（还原剂）
↓                                       ↓
Fe²⁺ ←──────────────────────────────── Fe³⁺
        还原剂(Fe, Cu, KI)
```

**关键方程式**：
- Fe²⁺ → Fe³⁺：2Fe²⁺ + Cl₂ = 2Fe³⁺ + 2Cl⁻
- Fe³⁺ → Fe²⁺：2Fe³⁺ + Fe = 3Fe²⁺；2Fe³⁺ + Cu = 2Fe²⁺ + Cu²⁺

#### Fe³⁺ 的检验
1. **KSCN溶液** → 血红色溶液：Fe³⁺ + 3SCN⁻ ⇌ Fe(SCN)₃
2. **NaOH溶液** → 红褐色沉淀：Fe³⁺ + 3OH⁻ = Fe(OH)₃↓

#### Fe(OH)₂的特殊性
- 白色沉淀，极易被氧化：4Fe(OH)₂ + O₂ + 2H₂O = 4Fe(OH)₃
- 颜色变化：白色 → 灰绿色 → **红褐色**
- 制备需**隔绝空气**

### 第二节 金属材料

#### 合金
- 合金熔点**低于**各组分金属
- 合金硬度一般**大于**各组分金属

#### 铁合金
- **生铁**：含C 2%~4.3%
- **钢**：含C 0.03%~2%（碳素钢、合金钢）
- **不锈钢**：含Cr、Ni

#### 铝 Al
- 银白色轻金属，地壳中含量最丰富的金属元素
- 4Al + 3O₂ --点燃→ 2Al₂O₃

#### ★ Al₂O₃ 和 Al(OH)₃ 的两性

| 物质 | 与酸反应 | 与碱反应 |
|------|----------|----------|
| Al₂O₃ | Al₂O₃ + 6H⁺ = 2Al³⁺ + 3H₂O | Al₂O₃ + 2OH⁻ = 2AlO₂⁻ + H₂O |
| Al(OH)₃ | Al(OH)₃ + 3H⁺ = Al³⁺ + 3H₂O | Al(OH)₃ + OH⁻ = AlO₂⁻ + 2H₂O |

> Al(OH)₃ 受热分解：2Al(OH)₃ --加热→ Al₂O₃ + 3H₂O

## 第四章 物质结构 元素周期律

> **地位**：高中化学理论基石，"位-构-性"是高考必考推理模型。

![第四章 物质结构 元素周期律](./knowledge_map_ch4.svg)

### 第一节 原子结构与元素周期表

#### 原子结构
- **原子序数 = 质子数 = 核电荷数 = 核外电子数**
- **质量数(A) = 质子数(Z) + 中子数(N)**
- 核素、**同位素**（¹H ²H ³H；¹²C ¹³C ¹⁴C）

#### 核外电子排布规律
- 电子层：K L M N O P Q（n=1~7）
- 每层最多 2n² 个电子
- 最外层不超过 8 个（K层不超过2个）
- Na(2-8-1)、Cl(2-8-7)、Al(2-8-3)

#### 元素周期表结构
- **7个周期**（横行）：1-3短周期，4-7长周期
- **16个族**（纵行）：7主族(A) + 7副族(B) + 0族 + VIII族
- **周期序数 = 电子层数**
- **主族序数 = 最外层电子数**

### 第二节 元素周期律

#### 同周期递变（左→右）

| 性质 | 变化趋势 |
|------|----------|
| 原子半径 | ↓ 逐渐减小 |
| 金属性 | ↓ 逐渐减弱 |
| 非金属性 | ↑ 逐渐增强 |
| 最高正化合价 | +1 → +7 |
| 最高价氧化物对应水化物 | 碱性↓ 酸性↑ |

#### 同主族递变（上→下）

| 性质 | 变化趋势 |
|------|----------|
| 原子半径 | ↑ 逐渐增大 |
| 金属性 | ↑ 逐渐增强 |
| 非金属性 | ↓ 逐渐减弱 |

#### 元素金属性/非金属性强弱判断

**金属性强**：①与水/酸反应越剧烈 ②最高价氧化物水化物碱性越强 ③置换反应
**非金属性强**：①与H₂化合越容易 ②气态氢化物越稳定 ③最高价含氧酸酸性越强 ④置换反应

#### ★ "位-构-性" 三角关系
```
     位置（周期表位置）
        /          \
       /            \
   结构（原子结构） —— 性质（元素性质）
```
三者互相推断，**知一推二**。

### 第三节 化学键

#### 离子键 vs 共价键

| 键型 | 形成方式 | 形成条件 | 示例 |
|------|----------|----------|------|
| 离子键 | 阴阳离子间静电作用 | 活泼金属+活泼非金属 | NaCl, MgO |
| 共价键 | 共用电子对 | 非金属原子间 | HCl, H₂O, CO₂ |

**共价键分类**：
- **非极性键**：同种原子间（H-H、Cl-Cl、O=O）
- **极性键**：不同原子间（H-Cl、C-O、N-H）

#### 电子式书写
- 原子电子式：标出最外层电子 ·
- 离子电子式：Na⁺、[∶Cl∶]⁻
- 注意 NH₄Cl：既含离子键又含共价键

#### 分子间作用力与氢键
- **范德华力**：分子间作用力，影响熔沸点
- **氢键**：H₂O、HF、NH₃ 分子间氢键 → 异常高的沸点

#### "五同"概念辨析

| 概念 | 定义 | 示例 |
|------|------|------|
| 同位素 | 质子数同、中子数异 | ¹H ²H ³H |
| 同素异形体 | 同种元素、不同单质 | O₂/O₃、金刚石/石墨 |
| 同系物 | 结构相似、组成差CH₂ | 烷烃系列 |
| 同分异构体 | 分子式同、结构异 | — |
| 同一物质 | 完全相同 | — |
