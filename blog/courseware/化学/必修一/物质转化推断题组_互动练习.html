第 1 题 基础
A、B、C 是三种常见物质。A 是自然界中含量最多的金属单质，B 是 A 的氧化物，C 是 A 与水反应的产物。
A（单质） +O₂→ B +H₂O→ C △→ ?
提示：地壳中含量最多的金属元素是铝
A = Al，B = Al₂O₃，C = Al(OH)₃，加热产物 = Al₂O₃
4Al + 3O₂ → 2Al₂O₃
Al₂O₃ + 3H₂O → 2Al(OH)₃（反应较缓慢，实际Al₂O₃难溶于水，此处指氢氧化铝的来源）
2Al(OH)₃ →(△) Al₂O₃ + 3H₂O
关键：Al(OH)₃ 既能与酸反应，又能与强碱反应，体现两性。加热分解为氧化物，是Al(OH)₃的特征反应。
第 2 题 基础
物质 A 是常见的活泼金属，与水剧烈反应生成 B（气体）和 C（溶液）。C 与过量 CO₂ 反应得到 D，D 加热得到 E，E 与 C 反应重新得到 C。
A +H₂O→ B↑ + C +CO₂(过量)→ D △→ E +C→ C
A=Na，B=H₂，C=NaOH，D=NaHCO₃，E=Na₂CO₃
2Na + 2H₂O → 2NaOH + H₂↑
NaOH + CO₂(过量) → NaHCO₃
2NaHCO₃ →(△) Na₂CO₃ + H₂O + CO₂↑
Na₂CO₃ + CO₂ + H₂O → 2NaHCO₃（不对，这里E+C→C应为：Na₂CO₃ + H₂O + CO₂ → 2NaHCO₃）
陷阱：NaOH与CO₂的比例决定产物！少量CO₂→Na₂CO₃；过量CO₂→NaHCO₃
第 3 题 中等
A 是黄绿色气体，B 是 A 溶于水的溶液，C 是强碱，A 与 C 反应得到 D 和 E 的混合液，D 加入盐酸得到 A，E 加热分解得到 C 和 A 以及 F（液体）。
A（黄绿气） +C(NaOH)→ D + E △→ C+A+F
A=Cl₂，B=盐酸/氯水，C=NaOH，D=NaCl，E=NaClO，F=H₂O
Cl₂ + 2NaOH → NaCl + NaClO + H₂O
NaClO + 2HCl → NaCl + Cl₂↑ + H₂O
2NaClO →(△) 2NaCl + O₂↑（注意：加热NaClO分解生成O₂，不是Cl₂）
纠错提醒：NaClO加热分解为NaCl和O₂，不会还原出Cl₂。题目中"E加热得到A（Cl₂）"表述有误——实际是NaClO与酸反应才能得到Cl₂。加热只得O₂。
第 4 题 中等
铁三角推断：单质 A 与 B（非氧化性酸）反应得到 C，C 加入 D（氧化性试剂）得到 E，E 加入 F（过量）得到 G（红褐色沉淀）。G 加热分解得 H（红棕色粉末）。
Fe +HCl→ C +Cl₂→ E +NaOH→ G↓ △→ H
C=FeCl₂，E=FeCl₃，G=Fe(OH)₃（红褐色），H=Fe₂O₃（红棕色）
Fe + 2HCl → FeCl₂ + H₂↑
2FeCl₂ + Cl₂ → 2FeCl₃
FeCl₃ + 3NaOH → Fe(OH)₃↓ + 3NaCl
2Fe(OH)₃ →(△) Fe₂O₃ + 3H₂O
记忆口诀：Fe遇非氧化性酸→+2价；遇强氧化剂（Cl₂、HNO₃稀）→+3价；Fe(OH)₂白色→Fe(OH)₃红褐色→Fe₂O₃红棕色
第 5 题 综合
A、B、C、D、E 均含同一种元素，A 是单质，B 是 A 在空气中燃烧的产物（不同于直接氧化），C 是 B 与水反应的产物，D 是 C 与 CO₂ 反应的产物，E 是另一种含该元素的盐。B 还可与 D 反应。
Na 点燃→ B +H₂O→ C+O₂ +CO₂→ D+O₂
关键：Na在空气中点燃生成Na₂O₂，不是Na₂O
B=Na₂O₂，C=NaOH，D=Na₂CO₃
2Na + O₂ →(点燃) Na₂O₂
2Na₂O₂ + 2H₂O → 4NaOH + O₂↑
2Na₂O₂ + 2CO₂ → 2Na₂CO₃ + O₂↑
高考高频考点：Na₂O₂与水、CO₂反应均放O₂，Na₂O₂是强氧化剂（可漂白）。Na₂O₂中O为-1价，属于过氧化物。
易错：Na在常温下氧化生成Na₂O；点燃才生成Na₂O₂！
第 6 题 综合·综合推断
物质 A、B、C、D、E 满足以下条件：A 为常见非金属单质，B 是 A 与 Fe 反应的产物（与 A+HCl 不同），C 是 B 溶液中加 KSCN 出现血红色的物质，D 是 B 加入足量 Fe 后的产物，E 是 D 与 NaOH 生成的白色沉淀，E 在空气中变色。
Cl₂ +Fe→ B KSCN检验 血红 +Fe(足量)→ D +NaOH→ E(白)→氧化→ 红褐
B=FeCl₃，D=FeCl₂，E=Fe(OH)₂（白色）→ 氧化为 Fe(OH)₃（红褐色）
2Fe + 3Cl₂ →(点燃) 2FeCl₃
2FeCl₃ + Fe → 3FeCl₂（足量Fe把Fe³⁺还原为Fe²⁺）
FeCl₂ + 2NaOH → Fe(OH)₂↓(白) + 2NaCl
4Fe(OH)₂ + O₂ + 2H₂O → 4Fe(OH)₃（白→灰绿→红褐）
检验区分：Fe²⁺加KSCN无颜色变化（再加氧化剂变红），Fe³⁺加KSCN立即血红。
综合考点：Cl₂氧化性强，Fe只能被氧化成+3价；足量Fe能把FeCl₃还原为FeCl₂。
