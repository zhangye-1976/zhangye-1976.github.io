第 1 题 基础
A、B、C、D 是含硫物质。A 是黄色固体单质，B 是 A 燃烧的产物，C 是 B 与 H₂O 反应的产物，D 是 C 与 O₂ 催化氧化的产物。
A（黄色固体） +O₂→ B +H₂O→ C +O₂催化→ D
提示：硫在纯氧中燃烧产生蓝紫色火焰
A=S，B=SO₂，C=H₂SO₃，D=H₂SO₄
S + O₂ →(点燃) SO₂（蓝紫色火焰）
SO₂ + H₂O ⇌ H₂SO₃（可逆，酸性氧化物）
2SO₂ + O₂ ⇌(催化剂,△) 2SO₃，SO₃ + H₂O = H₂SO₄
关键：SO₂是+4价硫，既有氧化性又有还原性；H₂SO₄(浓)有强氧化性、吸水性、脱水性。
第 2 题 基础
A 是空气中最多的气体单质，B 是 A 与 H₂ 在高温高压催化剂下合成的物质，B 与 O₂ 催化氧化得到 C，C 与 O₂ 反应得到 D，D 与 H₂O 反应得到 E（硝酸）。
A（N₂） +H₂→ B +O₂催化→ C +O₂→ D +H₂O→ E
提示：这是工业制硝酸的完整流程
B=NH₃，C=NO，D=NO₂，E=HNO₃
N₂ + 3H₂ ⇌(高温高压,催化剂) 2NH₃（合成氨，可逆）
4NH₃ + 5O₂ →(催化剂,△) 4NO + 6H₂O（工业制硝酸第一步）
2NO + O₂ = 2NO₂（无色→红棕色）
3NO₂ + H₂O = 2HNO₃ + NO（歧化反应，通入空气使NO循环）
记忆口诀：「一氧二氮三水酸，循环氧化产率高」
第 3 题 中等
A 是最简单的有机化合物（CH₄），在光照下与 Cl₂ 发生取代反应，第一步产物为 B（气体），最终完全取代产物为 C（液体，灭火剂型）。A 不完全燃烧生成 D（黑色固体）和 E（气体）。
A（CH₄） +Cl₂光照→ B 完全取代→ C
A（CH₄） 不完全燃烧→ D+E
提示：取代反应逐步进行，产物是混合物
B=CH₃Cl（一氯甲烷，气体），C=CCl₄（四氯化碳，液体），D=C（炭黑），E=CO
CH₄ + Cl₂ →(光) CH₃Cl + HCl（一氯甲烷，气体）
CH₃Cl + Cl₂ →(光) CH₂Cl₂ + HCl（二氯甲烷，液体）
最终：CHCl₃（氯仿），CCl₄（四氯化碳）
2CH₄ + 3O₂ →(不完全燃烧) 2CO + 4H₂O
易错点：甲烷取代产物是混合物，不是单一产物；CCl₄可作灭火剂（不支持燃烧）。
第 4 题 中等
A 是平面形分子，含 C=C 双键（所有原子共面），能使溴水褪色并生成 B（1,2-二溴乙烷）。A 在催化剂作用下与 H₂O 加成生成 C（工业制乙醇）。A 还能发生加聚反应生成 D（常用塑料）。
A（C₂H₄） +Br₂→ B +H₂O加成→ C 加聚→ D
提示：乙烯使酸性 KMnO₄ 褪色是氧化反应（与溴水原理不同）
B=CH₂Br−CH₂Br，C=CH₃CH₂OH（乙醇），D=−[CH₂−CH₂]−ₙ（聚乙烯）
CH₂=CH₂ + Br₂ → CH₂Br−CH₂Br（加成，溴水褪色）
CH₂=CH₂ + H₂O →(催化剂) CH₃CH₂OH（加成，工业制乙醇）
nCH₂=CH₂ →(催化剂) −[CH₂−CH₂]−ₙ（加聚，原子利用率100%）
鉴别：烷烃（CH₄）不能使溴水褪色；烯烃（C₂H₄）可以。乙烯与酸性KMnO₄是氧化反应，与Br₂是加成反应。
第 5 题 较难
A 是乙醇（CH₃CH₂OH），在 Cu 催化加热条件下被氧化为 B（有刺激性气味的液体），B further 被氧化为 C（弱酸，能与 A 发生酯化反应生成 D）。D 是有果香味的液体，难溶于水。酯化反应中，浓硫酸的作用是催化剂和吸水剂。
A（乙醇） Cu催化氧化→ B 氧化→ C 酯化→ D
提示：酯化反应机理是「酸脱羟基醇脱氢」
B=CH₃CHO（乙醛），C=CH₃COOH（乙酸），D=CH₃COOCH₂CH₃（乙酸乙酯）
2CH₃CH₂OH + O₂ →(Cu,△) 2CH₃CHO + 2H₂O（Cu丝：红→黑→红）
2CH₃CHO + O₂ →(催化剂) 2CH₃COOH
CH₃COOH + CH₃CH₂OH ⇌(浓H₂SO₄,△) CH₃COOCH₂CH₃ + H₂O
铜丝现象：红热Cu → 插入乙醇 → 变黑(CuO) → 又变红(Cu)，证明Cu作催化剂。
酯化机理：酸脱−OH，醇脱−H，生成H₂O。饱和Na₂CO₃溶液吸收乙醇、中和乙酸、降低酯溶解度。
第 6 题 综合
A、B、C、D 是含氮物质。A 是+4价氮的氧化物（红棕色），与水反应生成 B（+5价氮的酸）和 C（+2价氮的氧化物）。向 B 溶液中加入 Cu 片（稀溶液），生成 D（无色气体，遇空气变红棕色）。
A（红棕气） +H₂O→ B + C
B（HNO₃稀） +Cu→ D（无色） +空气→ 红棕
提示：NO₂与水反应是歧化反应；浓/稀HNO₃与Cu反应的还原产物不同
A=NO₂，B=HNO₃，C=NO，D=NO（稀HNO₃还原产物）
3NO₂ + H₂O = 2HNO₃ + NO（歧化：+4→+5 和 +4→+2）
Cu + 4HNO₃(浓) = Cu(NO₃)₂ + 2NO₂↑ + 2H₂O（浓→NO₂）
3Cu + 8HNO₃(稀) = 3Cu(NO₃)₂ + 2NO↑ + 4H₂O（稀→NO）
2NO + O₂ = 2NO₂（无色NO遇空气立即变红棕色，用于检验NO）
高考高频：浓HNO₃还原产物是NO₂（红棕）；稀HNO₃还原产物是NO（无色）。NO₂与水歧化生成NO，需通空气使NO重新氧化为NO₂以提高吸收率。
