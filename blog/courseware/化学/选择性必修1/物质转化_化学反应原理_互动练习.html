能量转化
第 1 题（基础）
已知：
① C(s) + O₂(g) = CO₂(g)　ΔH₁ = -393.5 kJ/mol
② CO(g) + ½O₂(g) = CO₂(g)　ΔH₂ = -283.0 kJ/mol
求 C(s) + ½O₂(g) = CO(g) 的 ΔH。
✅ 正确答案：B

解析（盖斯定律）：
目标反应 = ① - ②：
ΔH = ΔH₁ - ΔH₂ = (-393.5) - (-283.0) = -110.5 kJ/mol

★ 易错点：记住盖斯定律——反应热只与始态和终态有关，与途径无关。计算时注意符号！

第 2 题（基础）
写出 H₂(g) 的燃烧热化学方程式（燃烧热：285.8 kJ/mol，生成液态水）。下列写法正确的是？
✅ 正确答案：B

解析：
燃烧热定义：1 mol 纯物质完全燃烧生成稳定氧化物放出的热量。
- "稳定氧化物"：H₂O 必须是 液态（不是气态）→ 排除 A
- 系数是 1 mol H₂ 反应 → 排除 C
- 燃烧是放热反应，ΔH < 0 → 排除 D

正确答案：B
平衡转化
第 3 题（中等）
在一定温度下，反应 CO(g) + H₂O(g) ⇌ CO₂(g) + H₂(g) 的 K = 1.0。
起始时 [CO] = 2.0 mol/L，[H₂O] = 10 mol/L，求 CO 转化为 CO₂ 的转化率。
✅ 正确答案：C（83%）

解析（三段式）：
 COH₂OCO₂H₂
起始2.01000
变化-x-x+x+x
平衡2.0-x10-xxx
K = x² / [(2.0-x)(10-x)] = 1.0
→ x² = (2.0-x)(10-x) = 20 - 12x + x²
→ 12x = 20 → x = 5/3 mol
转化率 = (5/3) / 2.0 × 100% = 83%
平衡转化
第 4 题（中等）
对于可逆反应：N₂(g) + 3H₂(g) ⇌ 2NH₃(g)　ΔH < 0
下列说法正确的是？
✅ 正确答案：C

解析：
A ❌：ΔH < 0（放热），升温平衡逆向移动，K 减小，N₂转化率降低。
B ❌：催化剂不改变化学平衡，只加快达到平衡的时间，NH₃产率不变。
C ✅：正反应气体分子数减少（4 → 2），增大压强平衡正向移动，N₂转化率提高。
D ❌：增大 H₂ 浓度，平衡正向移动，但 H₂ 自身转化率降低（另一种反应物转化率才提高）。

★ 口诀：加大廉价反应物浓度，提高另一种反应物的转化率！
离子平衡转化
第 5 题（中等）
常温下，关于 0.1 mol/L Na₂CO₃ 溶液，下列关系正确的是？
✅ 正确答案：B、D（单选题选 B）

解析（三大守恒）：

电荷守恒：[Na⁺] + [H⁺] = [HCO₃⁻] + 2[CO₃²⁻] + [OH⁻]
→ C 选项漏了 CO₃²⁻ 的系数 2，❌

物料守恒：[Na⁺] = 2([H₂CO₃] + [HCO₃⁻] + [CO₃²⁻])
→ B 选项格式正确，✅

质子守恒：[OH⁻] = [H⁺] + [HCO₃⁻] + 2[H₂CO₃]
→ D 选项 H₂CO₃ 系数应为 2，❌

★ 比较大小：Na₂CO₃ 水解呈碱性，[OH⁻] > [H⁺]，但 CO₃²⁻ 会水解，所以 [Na⁺] > [CO₃²⁻] > [OH⁻] > [HCO₃⁻] > [H⁺]
离子平衡转化
第 6 题（中等）
用 AgNO₃ 标准溶液滴定 Cl⁻ 溶液，以 K₂CrO₄ 为指示剂。
下列说法正确的是？
✅ 正确答案：D（都正确）

解析：

A ✅：莫尔法（Mohr method）原理——Ksp(Ag₂CrO₄) > Ksp(AgCl)，Cl⁻ 先沉淀完全后，Ag⁺ 才与 CrO₄²⁻ 生成砖红色 Ag₂CrO₄ 指示终点。

B ✅：沉淀溶解平衡，[Ag⁺] = Ksp(AgCl) / [Cl⁻]，当 [Cl⁻] = 1.0×10⁻⁵ M（沉淀完全）时，可算出血清 [Ag⁺]。

C ✅：计算沉淀所需 [Ag⁺]：
- Cl⁻ 沉淀： [Ag⁺] = √(Ksp) = 1.34×10⁻⁵ M
- CrO₄²⁻ 沉淀： [Ag⁺] = √(Ksp/[CrO₄²⁻]) ≈ 更大值
→ Cl⁻ 先沉淀，✅

电化学转化
第 7 题（综合）
氢氧燃料电池（酸性介质），下列说法正确的是？
✅ 正确答案：D

解析（电极反应式书写）：

燃料电池：燃料在负极氧化，O₂ 在正极还原。

负极（氧化）：H₂ → 2H⁺ + 2e⁻
正极（还原）：O₂ + 4H⁺ + 4e⁻ → 2H₂O
（酸性介质用 H⁺ 配平）

电子流向：负极 → 外电路 → 正极
离子移动：H⁺ 向正极移动（阳离子→正极；阴离子→负极）

★ 陷阱：别把正负极反应式写反了！口诀："燃负氧"（燃料在负极发生氧化反应）。
电化学转化
第 8 题（综合 · 计算）
用惰性电极电解 CuSO₄ 溶液，电流为 2A，通电 1930 秒。
已知法拉第常数 F = 96500 C/mol，Cu 的摩尔质量 = 64 g/mol。
求阴极析出 Cu 的质量（单位：g）。
✅ 正确答案：B（1.28 g）

解析（电子守恒法）：

1. 算电量：Q = I × t = 2A × 1930s = 3860 C
2. 算电子物质的量：n(e⁻) = Q / F = 3860 / 96500 = 0.04 mol
3. 电极反应：Cu²⁺ + 2e⁻ → Cu，故 n(Cu) = 0.04 / 2 = 0.02 mol
4. 算质量：m(Cu) = 0.02 × 64 = 1.28 g

★ 电子守恒法通用步骤：
Q = I·t → n(e⁻) = Q/F → 根据电极反应式系数 → 算目标物质量
