## 二、核心题型与解题模板

### 题型1：二次函数综合

**标准式**：$f(x) = ax^2 + bx + c$（$a \neq 0$）

**三要素**
| 要素 | 公式 | 说明 |
|------|------|------|
| 对称轴 | $x = -\frac{b}{2a}$ | 最值点所在 |
| 顶点 | $\left(-\frac{b}{2a}, \frac{4ac-b^2}{4a}\right)$ | |
| 判别式 | $\Delta = b^2 - 4ac$ | 根的个数 |

**动轴定区间问题（高考热点！）**

> **解题模板**：设 $f(x)=ax^2+bx+c$ 在区间 $[m,n]$ 上求最值。
>
> 1. 算对称轴 $x_0 = -\frac{b}{2a}$
> 2. 比较 $x_0$ 与区间 $[m,n]$ 的位置关系：
>    - $x_0 < m$：$f(x)$ 在 $[m,n]$ 单调（a>0增/a<0减）
>    - $m \leq x_0 \leq n$：顶点在区间内，最值在顶点或端点
>    - $x_0 > n$：$f(x)$ 在 $[m,n]$ 单调（a>0减/a<0增）
> 3. 比较 $f(m), f(n), f(x_0)$ 得出最值

> **易错提醒**：开口方向和区间端点都要看！不要只看顶点。

### 题型2：指数与对数运算

**核心恒等式**

| 运算法则 | 公式 |
|----------|------|
| $a^m \cdot a^n = a^{m+n}$ | 同底相乘，指数相加 |
| $(a^m)^n = a^{mn}$ | 幂的幂，指数相乘 |
| $\log_a(MN) = \log_a M + \log_a N$ | 对数加法 |
| $\log_a\frac{M}{N} = \log_a M - \log_a N$ | 对数减法 |
| $\log_a M^n = n\log_a M$ | 对数幂法则 |

**换底公式**：$\log_a b = \frac{\log_c b}{\log_c a}$

**常用对数技巧**：
- $\log_a b \cdot \log_b a = 1$（互为倒数）
- $a^{\log_a N} = N$（恒等式）
- $\log_a 1 = 0$，$\log_a a = 1$

> **口诀**：真数相乘对数加，真数相除对数减；指数拉到对数前，换底公式渡难关。

### 题型3：指数函数与对数函数

| 对比维度 | $y = a^x$（a>1） | $y = \log_a x$（a>1） |
|----------|:---:|:---:|
| 定义域 | R | $(0,+\infty)$ |
| 值域 | $(0,+\infty)$ | R |
| 定点 | $(0,1)$ | $(1,0)$ |
| 单调性 | 递增 | 递增 |
| 渐近线 | y=0（x轴） | x=0（y轴） |
| 图像关系 | 关于 y=x 对称（互为反函数） |

**比较大小问题模板**

1. **底数相同比指数**：利用单调性
2. **指数相同比底数**：利用幂函数单调性
3. **底数指数都不同**：找中间量（通常是 0 或 1）

### 题型4：函数零点与方程

**零点存在定理**：若 $f(x)$ 在 $[a,b]$ 连续，且 $f(a) \cdot f(b) < 0$，则 $(a,b)$ 内至少有一个零点。

**解题策略**
| 方法 | 适用场景 |
|------|---------|
| 直接解方程 | 可解析求解 |
| 零点定理 | 证明存在零点 |
| 图像法 | $f(x)=0 \Leftrightarrow$ 两函数图像交点 |
| 二分法 | 逼近零点近似值 |

### 题型5：三角函数综合

**必须刻在脑子里的值**

| 角度 | 0° | 30° | 45° | 60° | 90° |
|------|:--:|:---:|:---:|:---:|:---:|
| sin | 0 | 1/2 | √2/2 | √3/2 | 1 |
| cos | 1 | √3/2 | √2/2 | 1/2 | 0 |
| tan | 0 | √3/3 | 1 | √3 | — |

**核心公式清单**

| 公式 | 表达式 |
|------|--------|
| 平方关系 | $\sin^2\alpha + \cos^2\alpha = 1$ |
| 商数关系 | $\tan\alpha = \frac{\sin\alpha}{\cos\alpha}$ |
| 诱导公式 | "奇变偶不变，符号看象限" |
| 和角公式 | $\sin(\alpha\pm\beta)=\sin\alpha\cos\beta\pm\cos\alpha\sin\beta$ |
| 倍角公式 | $\sin 2\alpha = 2\sin\alpha\cos\alpha$ |
|  | $\cos 2\alpha = \cos^2\alpha - \sin^2\alpha = 2\cos^2\alpha-1 = 1-2\sin^2\alpha$ |

**$y = A\sin(\omega x + \varphi)$ 图像变换**

| 参数 | 含义 | 对图像影响 |
|------|------|-----------|
| **A** | 振幅 | 纵向拉伸/压缩，|A|倍 |
| **$\omega$** | 角频率 | 横向压缩/拉伸，周期 $T=\frac{2\pi}{\omega}$ |
| **$\varphi$** | 初相 | 左右平移（左加右减） |

**图像变换顺序**
- $y = \sin x \to \sin(x+\varphi) \to \sin(\omega x+\varphi) \to A\sin(\omega x+\varphi)$
- 口诀：**先平移后伸缩！** 顺序不能乱！

> **易错提醒**：$\sin x \to \sin 2x \to \sin(2x+\pi/3)$ 和 $\sin x \to \sin(x+\pi/3) \to \sin(2x+\pi/3)$ 平移量不同！注意是先伸缩还是先平移。

## 三、高考大题命题方向

| 题型 | 涉及知识 | 常见设问 |
|------|---------|---------|
| **三角函数大题** | 恒等变换+图像性质+解三角形 | 求周期/最值/单调区间 |
| **函数与导数综合** | 函数性质+导数（选必） | 求单调区间/极值/证明不等式 |

## 四、函数解题通用心法

1. **定义域先行**：拿到函数题，先写出定义域！
2. **图像辅助**：能画图的尽量画图，数形结合
3. **特殊值验证**：用 0、1、-1 等快速检验
4. **分类讨论**：参数题情况要分全，不重不漏
5. **逆向思维**：已知性质求参数，从性质反推条件
