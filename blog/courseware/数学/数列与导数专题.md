---

## 第一部分：数列

### 一、数列基础框架

| 概念 | 等差数列 | 等比数列 |
|------|---------|---------|
| 定义 | $a_{n+1}-a_n = d$（常数） | $\frac{a_{n+1}}{a_n} = q$（常数，$q \neq 0$） |
| 通项 | $a_n = a_1 + (n-1)d$ | $a_n = a_1 q^{n-1}$ |
| 中项 | $2a_n = a_{n-1}+a_{n+1}$ | $a_n^2 = a_{n-1} \cdot a_{n+1}$ |
| 前n项和 | $S_n = \frac{n(a_1+a_n)}{2}$ | $S_n = \frac{a_1(1-q^n)}{1-q}$（$q\neq1$） |
|  | $= na_1 + \frac{n(n-1)}{2}d$ | $S_n = na_1$（$q=1$） |

> **和的性质**：$S_n, S_{2n}-S_n, S_{3n}-S_{2n}$ 也成等差/等比（等差公差 $n^2d$，等比公比 $q^n$）

---

### 二、求通项公式六大方法

| 方法 | 适用条件 | 核心操作 |
|------|---------|---------|
| **公式法** | 等差/等比 | 直接套公式 |
| **累加法** | $a_{n+1}-a_n = f(n)$ | $a_n = a_1 + \sum_{k=1}^{n-1} f(k)$ |
| **累乘法** | $\frac{a_{n+1}}{a_n} = f(n)$ | $a_n = a_1 \cdot \prod_{k=1}^{n-1} f(k)$ |
| **构造法** | $a_{n+1} = pa_n + q$ | 凑等比：$a_{n+1}+\lambda = p(a_n+\lambda)$ |
| **利用 $S_n$** | 已知 $S_n$ 与 $a_n$ 关系 | $n=1$ 时 $a_1=S_1$；$n\geq2$ 时 $a_n=S_n-S_{n-1}$ |
| **数学归纳法** | 猜出通项后 | ①验证n=1 ②假设n=k ③证明n=k+1 |

---

### 三、求前n项和五大方法

| 方法 | 适用数列 | 公式/操作 |
|------|---------|----------|
| **公式法** | 等差/等比 | 直接套 Sn 公式 |
| **分组求和** | 等差+等比混合 | 分开求和再相加 |
| **裂项相消** | 分式型（分母可拆） | $\frac{1}{n(n+1)} = \frac{1}{n} - \frac{1}{n+1}$ |
| **错位相减** | $\{a_n \cdot b_n\}$（等差×等比） | 乘公比后相减 |
| **倒序相加** | 首末等距项和有规律 | 正写+倒写求和 |

**裂项公式锦囊**

| 裂项 | 公式 |
|------|------|
| $\frac{1}{n(n+1)}$ | $\frac{1}{n} - \frac{1}{n+1}$ |
| $\frac{1}{n(n+k)}$ | $\frac{1}{k}\left(\frac{1}{n} - \frac{1}{n+k}\right)$ |
| $\frac{1}{(2n-1)(2n+1)}$ | $\frac{1}{2}\left(\frac{1}{2n-1} - \frac{1}{2n+1}\right)$ |
| $\frac{1}{\sqrt{n}+\sqrt{n+1}}$ | $\sqrt{n+1} - \sqrt{n}$ |

**错位相减模板**（$\{a_n\}$等差，$\{b_n\}$等比）

$S_n = a_1b_1 + a_2b_2 + \cdots + a_nb_n$

$qS_n = a_1b_2 + a_2b_3 + \cdots + a_{n-1}b_n + a_nb_{n+1}$

两式相减：$(1-q)S_n = a_1b_1 + d(b_2+b_3+\cdots+b_n) - a_nb_{n+1}$

> **口诀**：乘公比，错位减，等比求和中间段。

## 第二部分：导数

### 四、导数基础

**定义**：$f'(x_0) = \lim_{\Delta x \to 0} \frac{f(x_0+\Delta x)-f(x_0)}{\Delta x}$

**几何意义**：$f'(x_0)$ 是曲线在 $(x_0, f(x_0))$ 处的**切线斜率**。

**基本求导公式**

| 函数 | 导数 | 函数 | 导数 |
|------|------|------|------|
| $C$（常数） | 0 | $x^n$ | $nx^{n-1}$ |
| $\sin x$ | $\cos x$ | $\cos x$ | $-\sin x$ |
| $e^x$ | $e^x$ | $\ln x$ | $\frac{1}{x}$ |
| $a^x$ | $a^x \ln a$ | $\log_a x$ | $\frac{1}{x\ln a}$ |

**四则运算法则**

| 运算 | 公式 |
|------|------|
| 和差 | $(u \pm v)' = u' \pm v'$ |
| 积 | $(uv)' = u'v + uv'$ |
| 商 | $\left(\frac{u}{v}\right)' = \frac{u'v - uv'}{v^2}$ |

**复合函数求导（链式法则）**

$[f(g(x))]' = f'(g(x)) \cdot g'(x)$

> **口诀**：从外到内，层层求导，乘起来！

---

### 五、导数应用核心题型

#### 题型1：切线问题

**切线方程**：$y - f(x_0) = f'(x_0)(x - x_0)$

| 已知条件 | 方法 |
|----------|------|
| 切点已知 | 直接代入公式 |
| 切点未知（过某点） | 设切点 $(t, f(t))$，列方程求解 |
| 已知斜率 | $f'(x_0) = k$，解 $x_0$ |

> **易错**：过曲线上一点的切线和过曲线外一点的切线不是同一回事！

---

#### 题型2：单调性

$f'(x) > 0$ → $f(x)$ 递增；$f'(x) < 0$ → $f(x)$ 递减。

**求单调区间步骤**：
1. 求定义域
2. 求 $f'(x)$
3. 令 $f'(x)=0$，解得驻点
4. 列表讨论各区间 $f'(x)$ 的符号
5. 写出单调区间

---

#### 题型3：极值与最值

**极值判定**

| 条件 | 结论 |
|------|------|
| $f'(x_0)=0$ 且左侧 $f'>0$、右侧 $f'<0$ | 极大值点 |
| $f'(x_0)=0$ 且左侧 $f'<0$、右侧 $f'>0$ | 极小值点 |
| $f'(x_0)=0$ 但两侧同号 | 不是极值点！ |

**闭区间最值**：比较所有极值和端点值，取最大/最小。

---

#### 题型4：恒成立与证明不等式

**恒成立问题套路**

$f(x) \geq 0$ 恒成立 → 求 $f(x)_{min}$

| 方法 | 操作 |
|------|------|
| **直接法** | 直接求 $f(x)$ 的最小值（需要讨论参数） |
| **分离参数法** | 化为 $a \leq g(x)$（或 $\geq$），求 $g(x)$ 最值 |
| **端点效应** | 利用区间端点的函数值/导数值 |

**证明不等式的套路**
1. 构造辅助函数 $h(x) = f(x) - g(x)$
2. 求导分析 $h(x)$ 的单调性和最值
3. 证 $h(x) \geq 0$ 或 $h(x)_{min} \geq 0$

---

### 六、导数核心解题模板

#### 模板：含参函数单调性讨论

已知 $f(x)$ 含参数 $a$，讨论单调性。

**步骤**：
1. 求 $f'(x)$，通分因式分解
2. 令 $f'(x)=0$，解出根（根可能含参数）
3. **分类讨论**：
   - 若 $a=0$（退化情况）
   - 比较根的大小（根是否在定义域内）
   - 各区间内 $f'(x)$ 符号
4. 列表写出每种情况下的单调区间

> **关键**：讨论要"不重不漏"！画数轴辅助。

---

## 七、易错提醒清单

| 易错点 | 提醒 |
|--------|------|
| 等比求和忘讨论 q=1 | $q=1$ 时 $S_n=na_1$，不要用 $\frac{a_1(1-q^n)}{1-q}$！ |
| $S_n$ 求 $a_n$ 忘分 n=1 | $a_n = S_n - S_{n-1}$ 只对 $n\geq2$ 成立 |
| 裂项不彻底 | 裂项后检查通分是否还原 |
| $f'(x_0)=0$ 不一定是极值 | 还要看两侧符号！ |
| 复合函数求导漏层 | 链式法则：每层都要导！ |
| 闭区间最值忘比端点 | 极值和端点都要比！ |
