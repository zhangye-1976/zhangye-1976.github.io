## 模型一：板块模型（滑块-木板）⭐⭐⭐⭐⭐

### 模型描述

滑块放在木板上，施加力或给初速度，分析两者相对运动。

```
┌─────────────────┐  ← 木板 M（长L，与地面μ₂）
│  ┌───────┐      │
│  │ 滑块 m│  →F  │  ← 滑块（与木板μ₁）
│  └───────┘      │
└─────────────────┘
```

### 分析步骤

| 步骤 | 操作 |
|:---:|------|
| ① | 分别画出滑块和木板的受力分析图 |
| ② | 写出各自的牛顿第二定律方程 |
| ③ | 判断是否发生相对滑动（比较加速度） |
| ④ | 若相对滑动→分别列运动学方程 |
| ⑤ | 利用几何关系：滑块位移 - 木板位移 = L（滑出条件） |

### 核心方程

```
滑块：F - f₁ = m·a₁ （f₁ = μ₁mg）
木板：f₁ - f₂ = M·a₂ （f₂ = μ₂(m+M)g）
```

> 🔴 **判断滑块是否滑出：** x₁ - x₂ ≥ L 时滑块从木板上滑出
>
> ⚠ **常见误区：** 未判断相对滑动就胡乱套公式！

### 变式

| 变式 | 特点 |
|------|------|
| 地面光滑（μ₂=0） | f₂=0，木板仅受滑块摩擦力 |
| 给木板初速度 | 滑块由静止被带动 |
| 力F作用在木板上 | 受力分析对象互换 |
| 斜面上的板块 | 加入重力分力 mgsinθ |

## 模型二：传送带模型 ⭐⭐⭐⭐⭐

### 模型描述

物体放在运动中的传送带上，分析物体运动过程。

### 两类传送带

| 类型 | 关键点 |
|------|--------|
| **水平传送带** | 物体先加速，与传送带共速后匀速 |
| **倾斜传送带** | 加入 mgsinθ，注意摩擦力方向会反向 |

### 水平传送带分析流程

```
物体轻放在速度v的传送带上：
① 受力：f = μmg（摩擦力向右）→ a = μg
② 加速到v所需时间：t₁ = v/a = v/(μg)
③ 加速位移：x₁ = v²/(2μg)
④ 若 x₁ < 传送带长度L：之后匀速
   若 x₁ ≥ L：始终加速，末速 = √(2μgL)
```

### 倾斜传送带（重点）

| 角度条件 | 物体运动 |
|----------|----------|
| tanθ < μ | 放上后加速到共速，之后匀速（摩擦力足以平衡下滑力） |
| tanθ > μ | 放上后先加速到共速，之后继续加速下滑（摩擦力反向！） |

> 🔴 **超级必考：** tanθ > μ 时，共速后摩擦力方向反转→从滑动摩擦变为滑动摩擦但方向变了！
>
> ⚠ 物体与传送带共速是关键转折点，必须单独分析！

### 常见问题

| 问题类型 | 解法 |
|----------|------|
| 物体到达另一端时间 | 分段：加速时间 + 匀速时间 |
| 摩擦生热 | Q = f·Δx（相对位移！不是对地位移） |
| 电动机多消耗电能 | ΔE = 物体动能增量 + 摩擦生热 |

## 模型三：弹簧模型 ⭐⭐⭐⭐

### 核心规律

| 规律 | 公式 | 说明 |
|------|------|------|
| 胡克定律 | F = kx | x为形变量（伸长或压缩） |
| 弹性势能 | E_p = ½kx² | 与形变量平方成正比 |
| 回复力 | F = -kx | 简谐运动（弹簧振子） |

### 弹簧连接体

```
╔═══m₁═══[弹簧k]═══m₂═══→F

系统加速度：a = F/(m₁+m₂)
弹簧弹力：T = m₂·a = m₂F/(m₁+m₂)
```

### 弹簧临界问题

| 临界条件 | 判断 |
|----------|------|
| 刚好分离 | 弹簧恢复原长（弹力=0）且 a₁=a₂ |
| 刚好离开地面 | N=0，kx=mg |
| 速度最大 | a=0（合力为零处） |

> 🔴 **弹簧问题关键：** 找到"原长位置"和"平衡位置"是解题的两把钥匙。
>
> ⚠ 弹簧的力不能突变！与其他力（绳拉力可突变）不同。

## 模型四：斜面模型 ⭐⭐⭐⭐

### 基本斜面

```
物体在倾角θ斜面上：
  下滑力：F = mgsinθ
  正压力：N = mgcosθ
  摩擦力：f = μmgcosθ
  
  下滑条件：mgsinθ > μmgcosθ → tanθ > μ
```

### 斜面+外力

| 外力方向 | 正压力变化 |
|----------|-----------|
| 水平推物体靠斜面 | N = mgcosθ + Fsinθ（增大正压力） |
| 水平拉物体离斜面 | N = mgcosθ - Fsinθ（减小正压力） |
| 沿斜面向上拉 | N = mgcosθ（不变） |

### 斜面上的板块模型

将板块模型放到斜面上，四个力都需沿斜面方向和垂直斜面方向分解。

## 模型五：连接体模型 ⭐⭐⭐

### 轻绳连接

```
m₁ ——T—— m₂ →F

整体法：a = F/(m₁+m₂)
隔离法求T：T = m₁·a = m₁F/(m₁+m₂)
```

### 滑轮连接（阿特伍德机）

```
   ○（定滑轮）
  / \
m₁   m₂（m₁ > m₂）

a = (m₁-m₂)g/(m₁+m₂)（整体法）
T = 2m₁m₂g/(m₁+m₂)
```

## 🔴 力学综合题答题模板

> 适用：大题中出现多过程、多物体的力学综合问题

```
Step 1：选对象（整体or隔离？先整体求a，再隔离求内力）
Step 2：画受力（每个物体单独画，不遗漏任何一个力）
Step 3：列方程（F=ma → x方向/y方向分别列）
Step 4：找关联（连接体a相同 / 相对位移关系 / 绳长不变）
Step 5：解方程（先代数后数值，注意单位）
Step 6：检验（单位、符号、物理合理性）
```
