## 力学实验（5个必考）

### 实验一：研究匀变速直线运动 ⭐⭐⭐⭐⭐

| 项目 | 内容 |
|------|------|
| **器材** | 打点计时器（电磁/电火花）、纸带、小车、钩码、刻度尺 |
| **原理** | 纸带上相邻相等时间内的位移差 Δx = aT² |
| **关键操作** | 先接通电源，后释放小车 |

**数据处理：**

```
纸带：· · · · · · · · ·
      s1 s2 s3 s4 s5 s6

逐差法：a = [(s₆+s₅+s₄)-(s₃+s₂+s₁)]/(9T²)
（6段数据取全部，减小误差）

瞬时速度：v_n = (s_n + s_{n+1})/(2T)
```

### 实验二：探究弹簧弹力与形变量的关系

| 项目 | 内容 |
|------|------|
| **原理** | 胡克定律 F = kx |
| **数据处理** | 作F-x图 → 斜率 = k |
| **易错** | 弹簧自重影响（先挂砝码再测原长） |

> 🔴 弹簧长度 ≠ 形变量！用伸长量 = 挂后长度 - 原长。

### 实验三：探究两个互成角度的力的合成规律

| 项目 | 内容 |
|------|------|
| **原理** | 平行四边形定则（等效替代思想） |
| **操作** | 两次拉橡皮筋到同一点O |
| **易错** | O点位置不变、细绳与木板平行 |

### 实验四：探究加速度与力、质量的关系 ⭐⭐⭐⭐⭐

| 项目 | 内容 |
|------|------|
| **原理** | 控制变量法 a=F/m |
| **关键** | 平衡摩擦力（垫高木板一端，不挂重物时小车匀速下滑） |
| **条件** | 钩码质量 m ≪ 小车质量 M（近似认为T=mg） |

```
a-F图像：
  未平衡摩擦：图线不过原点（截距在a轴或F轴）
  钩码质量过大：图线向下弯曲（m不是远小于M）
```

### 实验五：验证机械能守恒定律

| 项目 | 内容 |
|------|------|
| **原理** | mgh = ½mv²（或 ΔE_p = ΔE_k） |
| **测量** | 下落高度h和瞬时速度v |
| **关键** | 重物质量适当大（减小空气阻力影响） |

> ⚠ 不需要测量质量m（等式两边约掉）——但如果让你"计算"动能则需要m。

## 电学实验（7个必考）

### 实验六：测定金属的电阻率 ⭐⭐⭐⭐⭐

| 项目 | 内容 |
|------|------|
| **原理** | ρ = RS/L = πd²R/(4L) |
| **测量R** | 伏安法（电流表内接/外接） |
| **测L** | 刻度尺（接入电路的有效长度） |
| **测d** | 螺旋测微器（千分尺） |

**电流表内接vs外接：**

| 接法 | 适用条件 | Rx测偏 |
|------|----------|:---:|
| **内接**（A在V里面） | R_x ≫ R_A（大电阻） | 偏大 |
| **外接**（V在A外面） | R_x ≪ R_V（小电阻） | 偏小 |

> 口诀：**大内偏大，小外偏小**（大电阻用内接、测量值偏大）

### 实验七：描绘小灯泡的伏安特性曲线

| 项目 | 内容 |
|------|------|
| **电路** | 分压式接法（必须从零开始调节） |
| **特点** | I-U曲线不是直线（温度↑→电阻↑） |
| **关键** | 电压从0开始扫描 |

### 实验八：测定电源的电动势和内阻 ⭐⭐⭐⭐⭐

| 项目 | 内容 |
|------|------|
| **原理** | 闭合电路欧姆定律 U = E - Ir |
| **数据处理** | 作U-I图 → 纵截距=E，斜率绝对值=r |
| **误差** | 外接法（电压表分流）→ E测偏小，r测偏小 |

### 实验九：练习使用多用电表

| 项目 | 内容 |
|------|------|
| **测电压** | 并联，红表笔接高电位 |
| **测电流** | 串联，红进黑出 |
| **测电阻** | 每次换挡必须先欧姆调零！ |
| **读数** | 电阻读数 = 表盘示数 × 倍率 |

> 🔴 **超级易错：** 欧姆表刻度不均匀，左密右疏；测电阻时待测电阻必须与电源断开！

### 实验十：探究影响感应电流方向的因素

| 项目 | 内容 |
|------|------|
| **器材** | 线圈、条形磁铁、灵敏电流计（指针偏转方向） |
| **操作** | 磁铁插入/拔出线圈，观察电流计偏转 |
| **结论** | 验证楞次定律 |

### 实验十一：探究变压器原副线圈电压与匝数的关系

| 项目 | 内容 |
|------|------|
| **器材** | 可拆变压器、交流电压表 |
| **原理** | U₁/U₂ = n₁/n₂（理想变压器） |
| **操作** | 原线圈接交流电，测副线圈电压 |

### 实验十二：用双缝干涉测量光的波长

| 项目 | 内容 |
|------|------|
| **原理** | Δx = Lλ/d（L为缝到屏距离，d为双缝间距） |
| **测量** | 用测量头测出n条亮纹间距→算出Δx |
| **关键** | 滤光片获得单色光 |

## 📊 通用数据处理方法

### 逐差法（纸带数据）

```
目的：充分利用数据，减小偶然误差
方法：偶数段 → 前后对半分，逐差求平均
```

### 图像法

| 图像 | 斜率含义 | 截距含义 |
|------|----------|----------|
| v-t图 | 加速度a | 初速度v₀ |
| F-x图 | 劲度系数k | — |
| U-I图（电池） | -r | 电动势E |
| a-F图 | 1/m | 摩擦力f |

## 🔴 常见实验误差分析

| 误差来源 | 影响 |
|----------|------|
| 打点计时器摩擦 | 测量a偏小 |
| 钩码质量不是远小于小车质量 | a-F图弯曲 |
| 未平衡摩擦力 | a-F图不过原点 |
| 电压表分流（测电池E/r） | E偏小，r偏小 |
| 电流表分压（测电阻内接） | R测偏大 |
