## 模块一：复合场问题 ⭐⭐⭐⭐⭐

### 1.1 三种场的对比

| 场 | 力 | 特点 | 是否做功 |
|----|-----|------|:---:|
| **重力场** | G=mg（竖直向下） | 恒力 | ✅ mgh |
| **电场** | F=qE（正电荷顺E方向） | 恒力（匀强电场） | ✅ qU |
| **磁场** | F=qvB（⊥v⊥B） | 变力（⊥v，只改方向） | ❌ 永不做功 |

### 1.2 常见复合场组合

| 组合 | 典型问题 | 解题关键 |
|------|----------|----------|
| **E+B** | 速度选择器 | v=E/B 时粒子直线通过 |
| **G+E+B** | 质谱仪/回旋加速器 | 洛伦兹力=向心力 |
| **G+E** | 带电粒子在电场中偏转 | 类平抛运动分析 |
| **G+B** | （少见，通常忽略重力） | 洛伦兹力=向心力 |

### 1.3 速度选择器

```
         ↑E（电场方向向下）
  → v →
         ↓B（磁场方向垂直纸面向里，洛伦兹力向上）

qvB = qE → v = E/B（与q、m无关！）
```

### 1.4 回旋加速器

| 参数 | 公式 |
|------|------|
| 最大速度 | v_max = qBR/m（R为D形盒半径） |
| 最大动能 | E_k = (qBR)²/(2m) |
| 回旋周期 | T = 2πm/(qB)（与v无关！） |
| 交变电场频率 | f = 1/T = qB/(2πm) |

### 1.5 质谱仪

| 步骤 | 内容 |
|:---:|------|
| ① 加速 | qU = ½mv² |
| ② 偏转 | r = mv/(qB) = √(2mU/q)/B |
| ③ 测r → 求m/q | m/q = B²r²/(2U) |

## 模块二：电磁感应综合 ⭐⭐⭐⭐⭐

### 2.1 标准的导体棒模型

```
  × × × × B（匀强磁场垂直纸面向里）
  × × × ×
  ───□───R───  ← 导轨（电阻不计）
  × |棒 × ×
  × v→ × ×
  ──────────
```

| 物理量 | 公式 | 说明 |
|--------|------|------|
| 感应电动势 | E = BLv | B⊥v⊥L |
| 感应电流 | I = E/(R+r) | r为棒电阻 |
| 安培力 | F安 = BIL = B²L²v/(R+r) | 方向用楞次/右手判断 |
| 电功率 | P = I²R = E²/(R+r) | |
| 焦耳热 | Q = I²Rt | |

### 2.2 两种驱动方式

| 驱动方式 | 过程 | 最终状态 |
|----------|------|----------|
| **恒力F驱动** | F - B²L²v/(R+r) = ma → a逐渐减小 | 匀速（F安=F时） |
| **初速度驱动** | -B²L²v/(R+r) = ma → a逐渐减小 | 静止（机械能→焦耳热） |

### 2.3 能量守恒法（最重要的方法！）

```
恒力F做功 = 动能变化 + 焦耳热
即：F·x = (½mv₂² - ½mv₁²) + Q

或：W_外 = ΔE_k + Q
```

> 🔴 **导体棒问题的万能钥匙：** 先用能量守恒求焦耳热，再用牛顿定律求运动细节。

### 2.4 感生电动势模型（磁场变化）

```
变化的B产生感生电场 → E = n·S·ΔB/Δt（面积不变）
变化的面积产生动生电动势 → E = n·B·ΔS/Δt（B不变）
```

## 模块三：交变电流与变压器 ⭐⭐⭐⭐

### 3.1 正弦交变电流核心公式

| 量 | 公式 | 关系 |
|----|------|------|
| 瞬时值 | e=E_m·sinωt | |
| 最大值 | E_m = nBSω | |
| 有效值 | E = E_m/√2 | 正弦波专用 |
| 平均值 | E_avg = 2E_m/π | 半周期 |

### 3.2 变压器"决定关系"（超级易错！）

```
┌───────┐    ┌───────┐
│  U₁   │    │  U₂   │
│  n₁   │════│  n₂   │──→负载R
│  I₁   │    │  I₂   │
└───────┘    └───────┘
```

| 决定方向 | 说明 |
|----------|------|
| U₁ → U₂ | U₂由U₁和匝数比决定（U₂=n₂/n₁·U₁） |
| I₂ → I₁ | I₂=U₂/R → I₁=n₂/n₁·I₂（负载决定电流！） |
| P₂ → P₁ | P₂=U₂I₂=U₂²/R → P₁=P₂（理想变压器） |

> 🔴 **关键：** 副线圈负载电阻变化→I₂变化→I₁跟着变！不是U₁决定一切。

### 3.3 远距离输电口诀

```
升压→I减小→P损=I²R减小→降压→用户

P损 = (P/U)²·R输电  （P为输电功率，U为输电电压）
```

## 🔴 高频易错

| 易错点 | 正确理解 |
|--------|----------|
| 洛伦兹力做功？ | ❌ 永不做功（⊥v） |
| 楞次定律"阻碍"=阻止？ | ❌ 是"阻碍变化"，不是"阻止" |
| 变压器I₁=I₂? | ❌ I₁/I₂=n₂/n₁ |
| 安培力方向？ | ❌ 不用右手，用**左手定则** |
| 感应电流方向？ | 楞次定律（右手定则是特例） |
