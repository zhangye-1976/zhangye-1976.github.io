# 代谢途径专题：光合作用与细胞呼吸

> 必修1《分子与细胞》第5章核心 · 高考必考大题素材

---

## 一、总体框架

```
                  ┌──────────────────────────────┐
                  │      光合作用（叶绿体）         │
                  │  6CO₂ + 12H₂O ─→ C₆H₁₂O₆ + 6O₂ + 6H₂O  │
                  │         光能→化学能             │
                  └───────────┬──────────────────┘
                              │ 有机物(葡萄糖)
                              ▼
                  ┌──────────────────────────────┐
                  │    细胞呼吸（细胞质基质+线粒体）  │
                  │  C₆H₁₂O₆ + 6O₂ ─→ 6CO₂ + 12H₂O  │
                  │         化学能→ATP             │
                  └──────────────────────────────┘

      光能 → 光合作用 → 有机物中化学能 → 呼吸作用 → ATP中活跃化学能 → 生命活动
```

**物质循环**：CO₂和H₂O在光合与呼吸之间循环  
**能量流动**：光能→化学能（有机物）→化学能（ATP）→各项生命活动

## 二、细胞呼吸（氧化分解有机物，释放能量）

### 2.1 有氧呼吸三阶段

| 阶段 | 场所 | 反应过程 | ATP | [H] | CO₂ | H₂O |
|:---:|------|------|:---:|:---:|:---:|:---:|
| I 糖酵解 | **细胞质基质** | C₆H₁₂O₆ → 2C₃H₄O₃(丙酮酸) + 4[H] | **2** | 4 | 0 | 0 |
| II 柠檬酸循环 | **线粒体基质** | 2C₃H₄O₃ + 6H₂O → 6CO₂ + 20[H] | **2** | 20 | 6 | 消耗6 |
| III 氧化磷酸化 | **线粒体内膜** | 24[H] + 6O₂ → 12H₂O | **34** | -24 | 0 | 产生12 |

**总反应式：**
```
C₆H₁₂O₆ + 6H₂O + 6O₂ ─酶→ 6CO₂ + 12H₂O + 能量(38ATP)
```

**关键考点速记：**
- H₂O 在哪里消耗？ → **第二阶段**（线粒体基质）
- O₂ 在哪里消耗？ → **第三阶段**（线粒体内膜）
- CO₂ 在哪里产生？ → **第二阶段**（线粒体基质）
- [H] 在哪里产生？ → 第一、二阶段都产生
- [H] 在哪里消耗？ → **第三阶段**（与O₂结合生成水）
- 葡萄糖在哪里分解？ → 不能进入线粒体！先在细胞质基质糖酵解

### 2.2 无氧呼吸

| 类型 | 生物举例 | 产物 | 场所 | ATP |
|------|---------|------|------|:---:|
| 酒精发酵 | 酵母菌、大多数植物（如水稻根） | C₂H₅OH + CO₂ | 细胞质基质 | 2 |
| 乳酸发酵 | 动物细胞、乳酸菌、马铃薯块茎、玉米胚 | C₃H₆O₃（乳酸） | 细胞质基质 | 2 |

**注意：**
- 无氧呼吸**只在细胞质基质**中进行
- 第一阶段与有氧呼吸完全相同（糖酵解）
- 第二阶段丙酮酸不进入线粒体，在细胞质基质被还原
- **[H]的去路**：有氧呼吸→交给O₂；无氧呼吸→交给丙酮酸

### 2.3 呼吸作用实验考点

**探究酵母菌细胞呼吸方式：**
- 有氧组：接气泵（NaOH洗气瓶吸收CO₂→验证空气中的CO₂已被吸收）
- 无氧组：密封放置一段时间（消耗瓶内O₂）
- 检测CO₂：澄清石灰水变浑浊 或 溴麝香草酚蓝溶液（蓝→绿→黄）
- 检测酒精：酸性重铬酸钾溶液（橙色→灰绿色）

## 三、光合作用（将无机物转化为有机物，储存能量）

### 3.1 光合色素

| 色素 | 颜色 | 滤纸条位置 | 主要吸收光 |
|------|:---:|:---:|------|
| 胡萝卜素 | 橙黄色 | 最上（溶解度最大） | **蓝紫光** |
| 叶黄素 | 黄色 | 第二 | **蓝紫光** |
| 叶绿素a | 蓝绿色 | 第三（含量最多） | 红光、蓝紫光 |
| 叶绿素b | 黄绿色 | 最下（溶解度最小） | 红光、蓝紫光 |

**口诀**：胡叶a b（从上到下），"胡爷爱b"

**提取与分离实验：**
- 提取：无水乙醇（溶解色素）+ 碳酸钙（保护叶绿素）+ 二氧化硅（研磨充分）
- 分离：纸层析法（层析液：石油醚+丙酮+苯）
- 画滤液细线：细、齐、直，重复2-3次

### 3.2 光反应与暗反应对比

| 比较项目 | 光反应 | 暗反应（卡尔文循环） |
|---------|--------|-------------------|
| **场所** | 类囊体薄膜 | 叶绿体基质 |
| **条件** | 光、色素、酶 | 酶（不需光但需光反应产物） |
| **物质变化** | ①H₂O→O₂+[H] ②ADP+Pi→ATP | ①CO₂固定(C₅+CO₂→2C₃) ②C₃还原→(CH₂O) ③C₅再生 |
| **能量变化** | 光能→活跃化学能（ATP、NADPH） | 活跃化学能→稳定化学能（有机物） |
| **产物** | O₂、ATP、[H](NADPH) | (CH₂O)、ADP、Pi、NADP⁺ |

**关键考点：**
- **O₂全部来自H₂O的光解**，与CO₂无关！（同位素¹⁸O标记实验）
- 暗反应"不需要光" ≠ "只能在黑暗中进行"——有光也可以！
- C₃还原需要光反应提供的**ATP和[H]（NADPH）**
- CO₂浓度骤降→C₃减少、C₅积累；光照骤停→C₃积累、C₅减少

### 3.3 光合作用的影响因素

| 因素 | 曲线特征 | 限制因素分析 |
|------|---------|------------|
| 光照强度 | 先升后平（光饱和点） | 饱和前：光强；饱和后：CO₂或温度 |
| CO₂浓度 | 先升后平（CO₂饱和点） | 饱和前：CO₂；饱和后：光强或温度 |
| 温度 | 钟形曲线（最适温度） | 影响酶活性 |

- **光补偿点**：光合速率 = 呼吸速率（净光合=0）
- **光饱和点**：光照再增，光合速率不再增大
- **总光合速率 = 净光合速率 + 呼吸速率**

## 四、光合作用与呼吸作用的比较与联系

| 比较项目 | 光合作用 | 有氧呼吸 |
|---------|---------|---------|
| **代谢类型** | 合成代谢（同化作用） | 分解代谢（异化作用） |
| **场所** | 叶绿体 | 细胞质基质 + 线粒体 |
| **条件** | 需光、色素、酶 | 需酶（有无光均可） |
| **物质变化** | 无机物→有机物 | 有机物→无机物 |
| **能量变化** | 光能→化学能（储能） | 化学能→ATP+热能（放能） |
| **ATP** | 光反应产生（用于暗反应） | 三个阶段都产生（用于各项生命活动） |
| **[H]来源与去路** | H₂O光解产生 → 还原C₃ | 葡萄糖分解产生 → 与O₂结合 |

**曲线分析类题型核心公式：**
- 呼吸速率测定：黑暗条件下CO₂释放量或O₂吸收量
- 净光合速率测定：光照下CO₂吸收量或O₂释放量
- 总光合速率 = 净光合速率 + 呼吸速率

**密闭容器中气体变化：**
- 持续光照：CO₂↓、O₂↑（总光合 > 呼吸时）
- 黑暗：CO₂↑、O₂↓（只有呼吸）
- 弱光低于补偿点：CO₂↑、O₂↓（呼吸 > 光合）

## 五、ATP——能量"货币"的连接作用

```
光合作用光反应        有氧呼吸（三个阶段）
     │                      │
     ▼                      ▼
  ADP + Pi ───────→ ATP
                    │
                    ▼
          主动运输、肌肉收缩
          生物发光、神经传导
          物质合成……
```

- ATP与ADP的相互转化**不是可逆反应**（催化两个方向的酶不同）
- 合成ATP的能量来源：
  - 植物：光能（光合作用）+ 化学能（呼吸作用）
  - 动物：化学能（呼吸作用）

## 六、高考常见综合题型

### 题型1：光合呼吸曲线分析
给出一条或两条曲线（CO₂吸收/释放 vs 光照强度），判断：光补偿点、光饱和点、呼吸速率、净光合速率。

### 题型2：密闭容器气体变化
给出不同条件下（光照/黑暗、不同温度）O₂或CO₂的变化数据，计算总光合、净光合。

### 题型3：同位素示踪
¹⁸O标记H₂O → O₂有放射性（证明O₂来自水）；¹⁴C标记CO₂ → (CH₂O)有放射性（追踪碳路径）

### 题型4：实验设计
探究光照强度/CO₂浓度/温度对光合作用的影响——控制变量法（单一变量原则）
