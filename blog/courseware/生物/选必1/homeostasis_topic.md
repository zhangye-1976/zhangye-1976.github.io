## 专题概览

本书5章的知识点不是孤立的——高考大题几乎都是**跨章节综合**。本专题将分散在各章的知识整合为**四大解题模型**，帮你应对最难的神经-体液-免疫综合题。

```
稳态调节综合题解题体系
├── 模型一：神经-体液-免疫调节网络（综合题之王）
├── 模型二：分级调节与反馈调节（"轴"式分析）
├── 模型三：血糖/体温/水盐调节联动（三调节大串联）
└── 模型四：免疫过程全景分析（体液免疫+细胞免疫+免疫失调）
```

## 模型一：神经-体液-免疫调节网络综合题

> **高考频率：** 每年必考，全国乙卷近5年出现4次  
> **难度：** ★★★★★  
> **分值：** 8~12 分（解答题）

### 1.1 三大调节系统的关系框架

```
         神经调节（最快，局部）
            ↓ 调控 ————→ 体液调节（较慢，广泛）
            │                │
            ├——→ 免疫调节 ←——┤
            │    （清除病原体） │
            ├——————————————┤
        大多数内分泌腺受中枢神经系统调控
              │
        （如：寒冷 → 下丘脑 → TRH → TSH → 甲状腺激素 ↑）
```

**关键认知：** 三大调节不是并列的，而是**神经调节居于主导地位**。

- 神经调节 → 调控内分泌腺（神经-体液联系）
- 神经调节 → 影响免疫系统（胸腺受神经支配）
- 体液调节 → 影响免疫功能（激素可调节免疫应答）
- 免疫系统 → 通过细胞因子等影响神经系统和内分泌系统

### 1.2 综合题六步答题模板

**典型题干：** "某人感染某病毒后，出现发热、咳嗽、乏力等症状，请分析机体的调节过程。"

```
步骤1【免疫启动】：病原体 → 吞噬细胞摄取处理 → 呈递抗原
                    ↓
步骤2【体液免疫】：B细胞受刺激 → 增殖分化为浆细胞 → 产生抗体 → 清除细胞外病毒
                    ↓
步骤3【细胞免疫】：T细胞增殖分化为细胞毒性T细胞 → 识别并裂解被感染细胞
                    ↓
步骤4【发热机制】：病原体毒素 → 刺激下丘脑体温调节中枢 →
                    甲状腺激素↑ → 代谢产热↑
                    肾上腺素↑ → 糖原分解产热↑
                    皮肤血管收缩 → 散热↓
                    ↓
步骤5【负反馈】：体温升高到一定程度 → 负反馈抑制产热 → 体温稳定在较高水平
                    ↓
步骤6【二次免疫】：记忆B细胞、记忆T细胞形成 → 再次感染时快速应答
```

### 1.3 真题实战

**题目：** 某人感染流感病毒后出现高热（39℃），请回答：

**(1) 流感病毒侵入人体后，机体启动________免疫和________免疫。**

> **答案：** 体液免疫、细胞免疫  
> **思路：** 病毒既有胞外阶段（体液免疫对胞外病毒）又有胞内阶段（细胞免疫裂解被感染细胞），所以两者都需要。

**(2) 体温升高到 39℃ 过程中，机体产热________散热；维持 39℃ 时，产热________散热。**

> **答案：** 大于、等于  
> **易错：** 升温过程中产热 > 散热（体温上升），维持高温时产热 = 散热（体温稳定但恒定点上移）。不是产热始终大于散热！

**(3) 退热过程中，机体通过________（填"增加"或"减少"）皮肤血流量和________（填"增加"或"减少"）汗腺分泌来散热。**

> **答案：** 增加、增加  
> **思路：** 散热方式 = 皮肤血管舒张（血流量↑）+ 汗腺分泌↑（蒸发散热↑）。

## 模型二：分级调节与反馈调节

> **涉及章节：** 第2章（神经分级调节）+ 第3章（体液分级调节）  
> **难度：** ★★★★

### 2.1 "轴"式分级调节模型

神经调节和体液调节都有分级结构，可以统一理解：

```
神经系统的分级调节：          体液（激素）的分级调节：
大脑皮层（最高级中枢）        下丘脑（调控中枢）
    ↓ 控制                        ↓ TRH/CRH/...
下丘脑/脑干/小脑（次级中枢）    垂体（中间枢纽）
    ↓ 控制                        ↓ TSH/ACTH/...
脊髓（低级中枢）               甲状腺/肾上腺皮质/性腺（靶腺体）
    ↓                             ↓
效应器                         靶细胞
```

**可以统一画为"轴"模型：**

```
下丘脑
  │ 释放激素（促XX释放激素）
  ↓
垂体
  │ 促激素（促XX激素）
  ↓
靶腺体
  │ 腺体激素（XX激素）
  ↓
靶细胞 → 生理效应
  │
  └──（负反馈）→ 抑制下丘脑和垂体 ←──
```

### 2.2 核心实例对照表

| 调节轴 | 下丘脑激素 | 垂体激素 | 靶腺体 | 靶腺激素 | 功能 |
|--------|----------|----------|--------|---------|------|
| **甲状腺轴** | TRH | TSH | 甲状腺 | 甲状腺激素 | 促进代谢和发育 |
| **肾上腺皮质轴** | CRH | ACTH | 肾上腺皮质 | 皮质醇 | 调节应激反应 |
| **性腺轴** | GnRH | FSH/LH | 性腺 | 性激素 | 调节生殖功能 |

### 2.3 负反馈调节 —— 高考三种考法

**考法1：分析缺碘导致甲状腺肿大的机制**

```
缺碘 → 甲状腺激素合成不足
    → 对下丘脑和垂体的负反馈抑制减弱
    → TSH分泌增多 → 甲状腺代偿性增生 → "大脖子病"
```

**考法2：推测切除垂体的影响**

```
切除垂体 → TSH缺失 → 甲状腺激素分泌↓
         → 但 TRH 分泌↑（因为负反馈消失，下丘脑不受抑制）
         → 所以：甲状腺激素↓、TRH↑
```

> 🔴 **区分点：** 切除垂体后，TSH消失（垂体不在了），甲状腺激素因失去刺激而降低。

**考法3：甲亢/甲减判断**

```
甲亢（甲状腺激素↑）：TRH↓、TSH↓（负反馈抑制增强）
甲减（甲状腺激素↓）：TRH↑、TSH↑（负反馈抑制减弱）
```

> **口诀：** "高的更高，低的更低"——这是错的！  
> **正确口诀：** "靶腺高，上级低（负反馈强）；靶腺低，上级高（负反馈弱）"

### 2.4 反馈调节在三大系统中的统一性

```
体液调节：甲状腺激素↑ → 抑制下丘脑和垂体（负反馈）
体温调节：体温偏高 → 散热↑、产热↓（负反馈）
血糖调节：血糖偏高 → 胰岛素↑ → 血糖↓（负反馈）
水盐调节：渗透压↑ → ADH↑ → 水重吸收↑ → 渗透压↓（负反馈）
免疫调节：抗原清除后 → 免疫应答减弱（负反馈）
```

**核心规律：** 稳态维持的核心机制就是**负反馈调节**。正反馈在正常生理中很少见（如分娩、排尿、凝血）。

## 模型三：血糖/体温/水盐调节大串联

> **涉及章节：** 第3章全部 + 第2章（神经对内分泌的调控）  
> **难度：** ★★★★

### 3.1 血糖调节 —— 四激素两来源三去路

**血糖来源（3条）：**
```
1. 食物中糖类消化吸收（消化系统）
2. 肝糖原分解（肝脏）
3. 非糖物质转化（肝脏 → 糖异生，如乳酸、甘油、氨基酸→葡萄糖）
```

**血糖去路（3条）：**
```
1. 氧化分解（细胞呼吸，产生CO₂+H₂O+能量）—— 最主要
2. 合成糖原（肝脏和肌肉）
3. 转化为非糖物质（脂肪、某些氨基酸等）
```

**四大调节激素：**

| 激素 | 来源 | 作用机制 | 效应 |
|------|------|----------|:---:|
| 胰岛素 | 胰岛B细胞 | 促进组织细胞摄取、利用、储存葡萄糖 | 血糖↓ |
| 胰高血糖素 | 胰岛A细胞 | 促进肝糖原分解、促进非糖物质→葡萄糖 | 血糖↑ |
| 肾上腺素 | 肾上腺髓质 | 促进肝糖原分解 | 血糖↑ |
| 糖皮质激素 | 肾上腺皮质 | 促进非糖物质→葡萄糖 | 血糖↑(长期) |

> 🔴 **关键：** 胰岛素是**唯一**降低血糖的激素（其余都是升血糖的）。

**血糖调节中的神经-体液联系：**
```
血糖↓ → 刺激下丘脑 → 交感神经兴奋 → 肾上腺髓质分泌肾上腺素↑ → 血糖↑
                              ↘ 胰岛A细胞分泌胰高血糖素↑ → 血糖↑
```
这是典型的**神经-体液调节**结合（神经调控内分泌）。

### 3.2 体温调节 —— "三途径产热、两途径散热"

**产热途径（3条）：**
| 途径 | 占比 | 控制方式 |
|------|:---:|----------|
| 骨骼肌战栗 | 主要（寒冷时） | 神经调节（躯体运动神经） |
| 肝脏代谢产热 | 安静时主要 | 体液调节（甲状腺激素↑） |
| 褐色脂肪分解 | 婴儿、冬眠动物 | 神经调节 |

**散热途径（2条）：**
| 途径 | 占比 | 控制方式 |
|------|:---:|----------|
| 皮肤毛细血管舒张 | 主要 | 神经调节 |
| 汗腺分泌（蒸发散热） | 高温时重要 | 神经调节 |

**完整调节链分析：**

```
          寒冷刺激
              ↓
        冷觉感受器
              ↓ (传入神经)
      下丘脑体温调节中枢
              ↓
    ┌─────────┼─────────┐
    ↓(传出)   ↓(体液)   ↓(传出)
 骨骼肌战栗  TRH→TSH   皮肤血管收缩
   产热↑     →甲状腺   汗腺分泌↓
            激素↑→产热↑  散热↓
```

> 🔴 **分清楚：** 寒冷时骨骼肌战栗是**神经调节**（反射弧），甲状腺激素↑是**体液调节**（分级调节轴）。这两条通路同时启动。

### 3.3 水盐调节 —— 渗透压的动态平衡

**调节过程：**

```
饮水不足/失水过多 → 细胞外液渗透压↑
                        ↓
          下丘脑渗透压感受器（感受刺激）
                        ↓
           下丘脑合成ADH → 垂体释放ADH
                        ↓
         肾小管和集合管重吸收水↑ → 尿量↓
                        ↓
              细胞外液渗透压↓（恢复正常）
```

```
饮水过多 → 细胞外液渗透压↓ → ADH分泌↓ → 水重吸收↓ → 尿量↑
```

> 🔴 **三步记忆：**
> 1. 渗透压感受器在**下丘脑**（不在垂体、不在肾脏）
> 2. ADH由**下丘脑合成**、**垂体释放**
> 3. ADH作用在**肾小管和集合管**（促进重吸收水）

### 3.4 三调节对比速览

| 调节类型 | 感受器 | 调节中枢 | 激素 | 效应器/靶器官 | 反馈类型 |
|----------|--------|----------|------|-------------|:---:|
| 血糖 | 下丘脑血糖感受区 | 下丘脑 | 胰岛素/胰高血糖素 | 肝、肌肉等组织 | 负反馈 |
| 体温 | 皮肤温度感受器 | 下丘脑 | 甲状腺激素/肾上腺素 | 骨骼肌、皮肤血管、汗腺 | 负反馈 |
| 水盐 | 下丘脑渗透压感受器 | 下丘脑 | ADH | 肾小管和集合管 | 负反馈 |

**三条共同规律：**
1. 调节中枢都在**下丘脑**（它是稳态调节的司令部）
2. 都是**神经-体液调节**结合
3. 都是**负反馈调节**维持稳态

## 模型四：免疫过程全景分析

> **涉及章节：** 第4章  
> **难度：** ★★★★

### 4.1 体液免疫 vs 细胞免疫 —— 协同作战全景图

```
                    病原体（抗原）
                        ↓
        ┌──────── 吞噬细胞摄取处理 ────────┐
        ↓                                  ↓
    呈递给T细胞                        呈递给T细胞
        ↓                                  ↓
    T细胞活化                          T细胞活化
   ↙        ↘                        ↙        ↘
分泌细胞因子   辅助B细胞           增殖分化      增殖分化
  ↕              ↓                  ↓           ↓
B细胞 ←────────┘              细胞毒性T细胞   记忆T细胞
  ↓                                ↓
增殖分化                       与靶细胞密切接触
  ↓                                ↓
┌─┴────────┐                 靶细胞裂解（凋亡）
↓          ↓                  释放出胞内病原体
浆细胞   记忆B细胞                 ↓
  ↓                        体液免疫接手清除
产生抗体
  ↓
与抗原结合→被吞噬细胞清除
```

> 🔴 **关键细节：**
> - T细胞在两种免疫中都有作用（B细胞活化必需T细胞的细胞因子协助）
> - 抗体只能对付**细胞外**的病原体
> - 细胞内的病原体（病毒、结核杆菌等）需要**细胞免疫**先裂解感染细胞，释放出病原体，再由**体液免疫**清除
> - 所以绝大多数病原体需要**两种免疫协同作战**

### 4.2 特异性免疫的时间线 —— 二次免疫

```
                初次免疫                          二次免疫
抗体浓度 ↑                          ┌─────────────
         │         ┌── 记忆B细胞      │    ╱
         │        ╱      ↓           │   ╱
         │   ╱───╱   保持休眠     应答│  ╱ (更快、更强)
         │  ╱                     快  │ ╱
         │ ╱                       速  │╱
         │╱                        ────┼───────────
    ─────┼─────────────────────────────┼──────→ 时间
         0     1      2              再次接触抗原
             (周)
```

**二次免疫的特点（三个"更"）：**
1. **更快** —— 潜伏期大大缩短（记忆细胞直接增殖分化）
2. **更强** —— 抗体浓度远高于初次免疫
3. **更专** —— 针对性更强

> 🔴 **易错：** 二次免疫后，浆细胞主要来自**记忆B细胞的增殖分化**，而不是B细胞直接受抗原刺激。记忆B细胞是"升级版"。

### 4.3 免疫失调三类判断口诀

```
过敏：再次接触才发病，过敏原非人人怕
自身免疫：敌我不分打自己，器官损伤最害怕
免疫缺陷：兵力不足守不住，HIV毁T最可怕
```

**详细对比：**

| 类型 | 免疫能力 | 作用对象 | 发生时机 | 举例 |
|------|:---:|------|----------|------|
| 过敏反应 | 过强 | 过敏原（非病原体） | 再次接触 | 荨麻疹、过敏性鼻炎 |
| 自身免疫病 | 过强 | 自身组织 | 持续 | 类风湿性关节炎、系统性红斑狼疮 |
| 免疫缺陷病 | 过弱 | — | — | 艾滋病（获得性）、先天性免疫缺陷 |

### 4.4 免疫相关高频判断

| 情境 | 判断 | 理由 |
|------|:---:|------|
| HIV攻击什么细胞？ | 辅助性T细胞 | 导致免疫功能几乎丧失 |
| 器官移植为什么要配型？ | 减少排斥反应 | 受者T细胞攻击移植器官（细胞免疫） |
| 为什么疫苗要打加强针？ | 强化二次免疫 | 增加记忆细胞数量和活性 |
| 为什么感冒好了不会再得同一种？ | 记忆细胞存在 | 二次免疫快速清除 |
| 浆细胞能不能再次增殖？ | 不能 | 浆细胞是高度分化的终末细胞，寿命短 |

## 模型五：植物激素调节 —— 五看生长素

> **涉及章节：** 第5章  
> **难度：** ★★★

### 5.1 生长素的"两重性"分析模型

**核心：** 生长素具有两重性——低浓度促进生长，高浓度抑制生长。

**三器官的最适浓度：**

```
根（最敏感）     芽            茎（最不敏感）
最适：10⁻¹⁰    最适：10⁻⁸    最适：10⁻⁴
  促进↑          促进↑          促进↑
  │  ╱╲         │    ╱╲       │        ╱╲
  │ ╱  ╲        │   ╱  ╲      │       ╱  ╲
  │╱    ╲       │  ╱    ╲     │      ╱    ╲
━┿━━━━━━━━    ━┿━━━━━━━━  ━┿━━━━━━━━━━━━
  │       ╲     │       ╲    │
  抑制          │        ╲   │
                │抑制(高浓度)  │
```

### 5.2 两道必考题

**题1：顶端优势**

```
顶芽 → 产生生长素 → 极性运输 → 侧芽处生长素浓度高 → 抑制侧芽生长
                                                    ↓
                                             去除顶芽 → 侧芽生长
```

> **实质：** 顶芽产生的生长素向下运输，在侧芽处积累→浓度过高→**抑制**侧芽生长。

**题2：根的向地性和茎的背地性**

```
横放植物：
       ┌─ 根(对生长素敏感)
       │   近地侧生长素浓度高 → 抑制 → 生长慢
       │   远地侧生长素浓度低 → 促进 → 生长快
       │   → 根向地弯曲生长（向地性）
       │
       └─ 茎(对生长素不敏感)
           近地侧生长素浓度高 → 促进更强 → 生长快
           远地侧生长素浓度低 → 促进较弱 → 生长慢
           → 茎背地弯曲生长（背地性）
```

> 🔴 **关键：** 根和茎对生长素浓度的**敏感度不同**，导致近地侧高浓度在根中抑制生长、在茎中促进生长。

### 5.3 五大植物激素对比

| 激素 | 作用 | 关系 |
|------|------|------|
| 生长素 | 促进细胞伸长、促进生根、顶端优势 | — |
| 赤霉素 | 促进细胞伸长、促进种子萌发 | 与脱落酸**拮抗**（萌发） |
| 细胞分裂素 | 促进细胞分裂、延缓衰老 | 与生长素**协同** |
| 脱落酸 | 抑制分裂、促进休眠和脱落 | 与赤霉素**拮抗** |
| 乙烯 | 促进果实成熟 | — |

> **口诀：** "赤霉长高细胞裂，脱落休眠乙烯熟。赤脱对萌发，生长细胞协。"

## 综合自测：跨章节8题

> 每题都综合至少两个章节，适合考试前检测。

**第1题**（第2章+第3章综合）  
寒冷环境中，人体会出现的生理变化是________。  
A. 皮肤血管舒张 B. 甲状腺激素分泌增加  
C. 汗腺分泌增加 D. 胰岛素分泌增加

> **答案：** B  
> **分析：** 寒冷→甲状腺激素↑→产热↑。A、C是散热途径，寒冷时应减少。D与体温调节无关（血糖调节）。

**第2题**（第2章+第3章综合）  
下列属于神经-体液调节的是________。  
A. 膝跳反射 B. 寒冷时骨骼肌战栗  
C. 血糖降低时胰高血糖素分泌增加 D. 进食后胰岛素分泌增加

> **答案：** C  
> **分析：** 血糖降低→下丘脑（神经）→胰岛A细胞分泌胰高血糖素↑→血糖↑（体液），这是典型的神经-体液调节。A是纯神经反射，D是纯体液调节（血糖浓度直接刺激胰岛B细胞）。B是纯神经调节。

**第3题**（第4章综合）  
下列关于特异性免疫的叙述，正确的是________。  
A. T细胞不参与体液免疫  
B. 记忆细胞在抗原刺激下可直接产生抗体  
C. 浆细胞可增殖分化为记忆细胞  
D. 二次免疫中记忆B细胞迅速增殖分化为浆细胞

> **答案：** D  
> **分析：** A错，T细胞分泌细胞因子辅助B细胞活化；B错，记忆细胞增殖分化→浆细胞→才能产生抗体；C错，浆细胞是终末分化细胞，不能继续增殖分化。

**第4题**（第3章综合）  
给正常小鼠注射一定量胰岛素后，小鼠出现低血糖症状。此时再注射________可迅速缓解症状。  
A. 胰高血糖素 B. 生理盐水 C. 胰岛素 D. 甲状腺激素

> **答案：** A  
> **分析：** 胰岛素导致低血糖→需要升血糖激素→胰高血糖素促进肝糖原分解→血糖↑。

**第5题**（第2章+第3章+第4章综合）  
某人接受器官移植后，需要长期服用免疫抑制剂，其主要目的是________。  
A. 增强免疫功能 B. 抑制体液免疫  
C. 抑制T细胞对移植器官的攻击 D. 促进伤口愈合

> **答案：** C  
> **分析：** 器官移植的排斥反应主要是受者T细胞识别移植器官为"非己"抗原→细胞免疫攻击。免疫抑制剂抑制T细胞活性。

**第6题**（第5章综合）  
将植物幼苗横放，一段时间后根向地弯曲生长，茎背地弯曲生长。关于此现象的叙述，错误的是________。  
A. 重力使近地侧生长素浓度高于远地侧  
B. 根对生长素更敏感，近地侧高浓度抑制生长  
C. 茎对生长素不敏感，近地侧高浓度促进生长更强  
D. 根和茎中生长素的运输方向相同

> **答案：** D  
> **分析：** 生长素在根和茎中都是极性运输（形态学上端→下端），方向相同。D其实是对的，所以... 等等，让我重新审视。四个选项：A正确（重力导致），B正确，C正确，D也是正确的（都是极性运输）。这题应该是选D吗？——不，D是正确的，生长素在根和茎中都是极性运输。那这道题四个选项全对？让我换一下……

> *此题撤回重新设计：*

**第6题（新）**（第5章综合）  
将植物幼苗横放，根向地弯曲、茎背地弯曲的直接原因是________。  
A. 向光性 B. 生长素分布不均匀  
C. 顶端优势 D. 赤霉素的作用

> **答案：** B  
> **分析：** 重力→生长素在近地侧积累→分布不均匀→不同器官敏感度不同→弯曲生长。

**第7题**（第2章+第4章综合）  
关于免疫系统和神经系统的关系，正确的是________。  
A. 免疫系统不受神经系统调控  
B. 胸腺是免疫器官也是内分泌器官  
C. 淋巴因子可作用于神经系统  
D. 以上都正确

> **答案：** C  
> **分析：** A错，免疫受神经调控（如应激影响免疫）。B错，胸腺是免疫器官（T细胞成熟场所）但不是内分泌器官。C正确，某些细胞因子可影响神经系统。

**第8题**（全书综合——高考难度）  
健康人进入寒冷环境后，机体调节的叙述中正确的是________。  
A. 皮肤温度感受器感受寒冷刺激，属于体液调节  
B. 甲状腺激素和肾上腺素协同促进产热  
C. 皮肤毛细血管收缩属于体液调节  
D. 体温维持恒定说明产热和散热均停止

> **答案：** B  
> **分析：**  
> A：感受器→传入神经→中枢，是神经调节的起始，不是体液调节 ✗  
> B：甲状腺激素和肾上腺素都能促进产热，两者是协同作用 ✓  
> C：毛细血管收缩是神经调节（交感神经），不是体液 ✗  
> D：体温恒定说明产热=散热，不是都停止 ✗
