## 模块一：能量流动定量计算

> **高考频率：** 全国乙卷每年必考 1 道计算题  
> **难度：** ★★★★  
> **分值：** 4~6 分（通常在解答题第2~3问）

### 1.1 核心概念：同化量 ≠ 摄入量

这是能量计算题**最容易出错的地方**：

```
摄入量（吃进去的）
    │
    ├── 粪便（未被同化）→ 属于上一营养级的同化量！
    │
    └── 同化量（真正吸收的）
            │
            ├── 呼吸作用散失（热能，约 60%~70%，不可再利用）
            │
            ├── 用于生长、发育、繁殖
            │       │
            │       ├── 流入下一营养级
            │       ├── 流向分解者（遗体、遗骸）
            │       └── 未被利用（如活体中的能量）
            │
            └── 其他（死亡遗体的能量→分解者）
```

> 🔴 **最易错：** 粪便中的能量 → **不是**该营养级同化的，而属于**上一营养级**的同化量。  
> 比如：兔吃草，兔粪便中的能量 = 草的同化量（不是兔的）。

**核心公式：**
```
同化量 = 摄入量 − 粪便量
同化量 = 呼吸作用散失量 + 用于生长、发育、繁殖的能量
```

### 1.2 传递效率计算 —— 林德曼定律

**公式：**
```
传递效率 = (下一营养级同化量 ÷ 上一营养级同化量) × 100%
一般范围：10% ~ 20%
```

**典型例题1（基础）：**

> 某生态系统中，生产者固定的总能量为 1000 kJ，初级消费者同化 150 kJ，次级消费者同化 22.5 kJ。  
> 求：(1) 生产者→初级消费者的传递效率  
> (2) 初级消费者→次级消费者的传递效率

```
(1) 传递效率 = 150 ÷ 1000 × 100% = 15%（在 10%~20% 范围内 ✓）
(2) 传递效率 = 22.5 ÷ 150 × 100% = 15%
```

**典型例题2（易错）：**

> 某草原生态系统中，初级消费者（兔）摄入草中的能量为 500 J，兔呼吸消耗 180 J，粪便中的能量为 120 J，兔的遗体被分解者利用 50 J。  
> 求：兔的同化量是多少？流入次级消费者的能量最多是多少？

```
第一步：兔的同化量 = 摄入量 − 粪便量 = 500 − 120 = 380 J

第二步：兔用于生长、发育、繁殖的能量
     = 同化量 − 呼吸消耗量 = 380 − 180 = 200 J

第三步：流入次级消费者的能量最多
     = 同化量 × 20%（最大传递效率）= 380 × 20% = 76 J

⛔ 注意：不能用"用于生长、发育、繁殖的能量 200 J × 20% = 40 J"——这是错的！
因为"流入下一营养级"的能量 = 上一营养级同化量 × 传递效率，
不是"用于生长、发育、繁殖的能量 × 传递效率"。
"用于生长、发育、繁殖的能量"已经不含呼吸消耗了。
```

> 🔴 **超级易错陷阱：**  
> 很多同学看到"流入次级消费者的能量"就想用"用于生长的 200 J × 20%"，  
> 但正确是用**同化量 380 J × 20%**。因为"流入下一营养级"的定义是从**同化量**按传递效率算的，不是从"余量"算！

### 1.3 "最多/最少"问题速查表

这是能量计算中最具区分度的题型，必须熟练：

| 问题类型 | 计算规则 | 例题 |
|----------|----------|------|
| 求最高营养级**最多**获得多少能量 | 用**最长**食物链 × **10%**（每级） | 第3营养级 → 生产者 × 10% × 10% = × 1% |
| 求最高营养级**最少**获得多少能量 | 用**最短**食物链 × **20%**（每级） | 第3营养级 → 生产者 × 20% × 20% = × 4% |
| 求**最多需要**消耗多少生产者 | 用**10%**（每级）向后推 | 第3营养级增 1kg → 需生产者 1÷10%÷10% = 100kg |
| 求**最少需要**消耗多少生产者 | 用**20%**（每级）向后推 | 第3营养级增 1kg → 需生产者 1÷20%÷20% = 25kg |

> **口诀：** "最高最多，长链 10%（粗吃细算）—— 求最高营养级获得能量'最多'时按最长的链和10%算"
>
> 更简洁的记法：  
> - **"求获得，最多→长链10%，最少→短链20%"**  
> - **"求消耗，最多→10%，最少→20%"**

### 1.4 多食物链加权计算

当最高营养级通过多条食物链获取能量时：

**例题：**

```
食物网：
   草 → 兔 → 狐
   草 → 鼠 → 狐
   草 → 虫 → 鸟 → 狐

已知狐增重 1kg，且从兔和鼠中获得的能量各占 50%。
传递效率按 20% 计算。求至少需要草多少 kg？

解法：
狐从兔获得 0.5kg → 需要兔 0.5÷20%=2.5kg → 需要草 2.5÷20%=12.5kg
狐从鼠获得 0.5kg → 需要鼠 0.5÷20%=2.5kg → 需要草 2.5÷20%=12.5kg
总需要草 = 12.5 + 12.5 = 25kg
```

### 1.5 综合大题模型

**高考典型题：**

> 某湖泊生态系统中，生产者（浮游植物）固定的总能量为 2.4×10⁵ kJ。  
> 初级消费者（浮游动物）同化 3.6×10⁴ kJ，次级消费者（小鱼）同化 5.4×10³ kJ，  
> 三级消费者（大鱼）同化 8.1×10² kJ。  
>
> (1) 画出该生态系统的能量金字塔。  
> (2) 计算各营养级间的传递效率。  
> (3) 分析能量传递效率低的原因。

```
(1) 能量金字塔（数值）：
        8.1×10² kJ   ← 三级消费者（大鱼）
       5.4×10³ kJ    ← 次级消费者（小鱼）
      3.6×10⁴ kJ     ← 初级消费者（浮游动物）
     2.4×10⁵ kJ      ← 生产者（浮游植物）

(2) 传递效率：
    生产者→初级消费者：3.6×10⁴ ÷ 2.4×10⁵ × 100% = 15%
    初级→次级：5.4×10³ ÷ 3.6×10⁴ × 100% = 15%
    次级→三级：8.1×10² ÷ 5.4×10³ × 100% = 15%

(3) 传递效率低的原因：
    ①各营养级的呼吸作用消耗大量能量（以热能散失，约60%~70%）
    ②部分能量随遗体、粪便等流向分解者
    ③部分能量未被利用（如活体生物中的能量）
    ④食物链越长，顶级消费者获得的能量越少
```

## 模块二：食物网分析

> **难度：** ★★★  
> **题量：** 每年选择或解答中必出现

### 2.1 食物链的计数方法

**法则：**
1. 从**生产者**出发
2. 到**最高营养级**结束
3. 逐一列举，不遗漏、不重复

**例题：**

```
食物网：
        草 → 鼠 → 狐
        ↓    ↘
        虫 → 鸟 → 狐
              ↘
               蛇 → 鹰

分析：此食物网有 4 条食物链

① 草 → 鼠 → 狐                    （第3营养级：狐）
② 草 → 鼠 → 鸟 → 狐               （第4营养级：狐）
③ 草 → 虫 → 鸟 → 狐               （第4营养级：狐）
④ 草 → 虫 → 鸟 → 蛇 → 鹰          （第5营养级：鹰）
```

### 2.2 营养级的确定

**关键：** 同一生物在不同的食物链中可能属于**不同的营养级**。

| 生物 | 在食物链①的营养级 | 在食物链②的营养级 | 在食物链④的营养级 |
|------|:---:|:---:|:---:|
| 鼠 | 第2 | 第2 | — |
| 鸟 | — | 第3 | 第3 |
| 狐 | 第3 | 第4 | — |

> **规则：** 从生产者（第1营养级）开始数，数到目标生物是第几个环节就是第几营养级。

### 2.3 生物数量变化的连锁分析

**核心原则：**  
- 天敌↓ → 被捕食者↑（短期）；但被捕食者↑ → 天敌↑（长期），趋于平衡  
- 猎物↓ → 捕食者↓（尤其是专一性捕食者）

**分析框架（以经典食物网为例）：**

```
草 → 鼠 → 狐
草 → 虫 → 鸟 → 狐

【情景1】鼠大量减少
→ 狐的食物之一减少 → 狐的数量短期内减少
→ 但鸟仍可作为狐的食物 → 狐可能转向捕食鸟 → 鸟减少
→ 虫的天敌减少 → 虫增加 → 草减少

【情景2】狐大量减少
→ 鼠和鸟的天敌减少 → 鼠和鸟数量增加（短期）
→ 鼠增加 → 吃更多草 → 草减少
→ 鸟增加 → 吃更多虫 → 虫减少
```

**🔴 重要：** 分析时要说清楚"短期效应"还是"长期稳定状态"。

### 2.4 食物网的复杂性与稳定性的关系

```
食物网越复杂 → 营养结构越复杂 → 抵抗力稳定性越高
                                    ↓
                         某物种消失后，其他物种可替代其功能
                         即"冗余"越大，抗干扰能力越强
```

反之：食物网越简单 → 某关键物种消失 → 可能整个系统崩溃。

## 模块三：生态金字塔

> **难度：** ★★  
> **题型：** 选择题识图 + 解答题绘制

### 3.1 三种金字塔对比

| 金字塔类型 | 含义 | 形状特点 | 是否可以倒置 |
|------------|------|----------|:---:|
| **能量金字塔** | 各营养级的能量（同化量） | **永远是正金字塔** | ❌ 不可能倒置 |
| **数量金字塔** | 各营养级的个体数目 | 一般正金字塔 | ✅ 可能倒置（如一棵树上大量昆虫） |
| **生物量金字塔** | 各营养级的生物量（干重） | 一般正金字塔 | ✅ 可能倒置（如海洋浮游植物被大量浮游动物消耗） |

> 🔴 **关键：** 只有**能量金字塔不可能倒置**——因为能量逐级递减（10%~20%）是铁的规律。

### 3.2 能量金字塔的绘制步骤

```
步骤1：收集各营养级的同化量数据
步骤2：选择适当的比例（通常用矩形面积表示能量大小）
步骤3：从生产者（底层）开始，逐级向上堆叠
步骤4：标注营养级名称和能量数值

注意事项：
- 矩形宽度一致（比较面积 = 比较高度）
- 各层按比例缩放
- 生产者在最底层，顶级消费者在最顶层
```

### 3.3 三类金字塔速览示意

```
能量金字塔（不可倒置）：          生物量金字塔（可能倒置）：
     —— 三级消费者                    —— 浮游动物
   ————— 次级消费者                   —————— 浮游植物
 ———————— 初级消费者                （浮游植物生长快、周转快，
———————————————— 生产者              现存生物量可能小于消费者）
```

## 模块四：碳循环与环境热点

> **难度：** ★★★  
> **热点指数：** ★★★★★（碳中和是近年高频考点）

### 4.1 碳循环完整图解

```
                       大气中的 CO₂
                    ↗   ↑    ↖      ↖
          光合作用╱     │      ╲       ╲化石燃料
                ╱       │        ╲       ╲燃烧
               ↙        │          ↘       ↘
        生产者(有机物)    │          分解者   化石燃料
              │         │            │      (煤、石油、天然气)
              │摄食     │呼吸作用     │分解
              ↓         │            ↓
        消费者(有机物)──┘      动植物遗体、粪便等
              │
              │呼吸作用
              ↓
         大气中的 CO₂
```

### 4.2 碳循环中的"箭头"判断技巧

在碳循环图解选择题中，快速判断：

| 箭头方向 | 过程 | 箭头的化学本质 |
|----------|------|----------------|
| 大气CO₂ → 生产者 | 光合作用/化能合成 | C从无机→有机 |
| 生产者 → 大气CO₂ | 呼吸作用 | C从有机→无机 |
| 生产者 → 消费者 | 摄食（捕食） | 有机物传递 |
| 消费者 → 大气CO₂ | 呼吸作用 | C从有机→无机 |
| 生物 → 分解者 | 遗体/粪便中的有机物 | 有机物传递 |
| 分解者 → 大气CO₂ | 分解作用（实质也是呼吸） | C从有机→无机 |
| 化石燃料 → 大气CO₂ | 燃烧 | C从有机→无机 |

> **口诀：** "进群落靠光合，出群落靠呼吸。化石燃烧是高速通道（人为加速了碳循环）。"

### 4.3 碳中和综合题

**碳中和 = 碳排放量 = 碳吸收量**

```
碳排放（4大来源）：
1. 化石燃料燃烧（最主要）
2. 工业生产过程（如水泥生产）
3. 农业生产（如水稻田CH₄、反刍动物）
4. 土地利用变化（如森林砍伐）

碳吸收（2大途径）：
1. 植物光合作用（碳汇，最主要）
2. 海洋吸收 CO₂

实现碳中和的途径：
1. 能源结构调整（发展风能、太阳能、核能，减少化石燃料依赖）
2. 提高能源效率（节能）
3. 增加碳汇（植树造林、保护森林和草原、湿地恢复）
4. 碳捕获利用与封存（CCUS技术）
5. 改变生产生活方式（绿色低碳）
```

### 4.4 能量流动 vs 物质循环 —— 在出题中的关联

| 对比维度 | 能量流动 | 物质循环 |
|----------|----------|----------|
| 流动方向 | 单向（不可逆） | 循环（可往复） |
| 是否递减 | 逐级递减（10%~20%） | 不递减 |
| 最终归宿 | 以热能形式散失 | 回归无机环境后再被利用 |
| 范围 | 各营养级之间 | 生物圈（全球性） |

**综合题的纽带：**  
> "能量是物质循环的**动力**，物质是能量的**载体**。"  
> —— 高考最喜欢考这句话的填空题！

## 模块五："最值"问题解题技巧总汇

### 5.1 能量计算题基本分类

```
能量计算题
├── 类型A：已知上一营养级 → 求下一营养级获得能量
│   方法：上一营养级同化量 × 传递效率（10%或20%）
│
├── 类型B：已知下一营养级 → 求需要多少上一营养级
│   方法：下一营养级同化量 ÷ 传递效率（10%或20%）
│
├── 类型C：多食物链 + 比例分配
│   方法：按比例分别计算，再求和
│
└── 类型D：给定范围（最多/最少）
    方法：用"最值口诀"快速判断用10%还是20%
```

### 5.2 最值口诀速记表

| 已知条件 | 所求 | 用10%还是20% | 例子 |
|---------|------|:---:|------|
| 生产者能量已知 | 最高营养级获得**最多**能量 | 20% | 1000→200→40 |
| 生产者能量已知 | 最高营养级获得**最少**能量 | 10% | 1000→100→10 |
| 最高营养级增量已知 | **最少**消耗生产者 | 20%（高→低用÷） | 增1kg←5kg←25kg |
| 最高营养级增量已知 | **最多**消耗生产者 | 10%（高→低用÷） | 增1kg←10kg←100kg |

> **终极口诀：**  
> "求获得→顺箭头：最多用20%，最少用10%"  
> "求消耗→逆箭头：最多用10%（÷），最少用20%（÷）"

### 5.3 综合大题精练

**题目：** 某生态系统食物网如下：

```
玉米 → 鸡 → 人
玉米 → 人
玉米 → 牛 → 人
```

已知：人的食物来源中，1/3 来自鸡、1/3 来自牛、1/3 直接来自玉米。  
传递效率按 20% 计算。人增重 3 kg，**至少**需要玉米多少 kg？

```
【分析】"至少"→ 求消耗最少 → 用 20% 传递效率

(1) 人通过鸡获得的 1 kg（3 × 1/3 = 1 kg）
    需要鸡：1 ÷ 20% = 5 kg
    需要玉米：5 ÷ 20% = 25 kg
    
(2) 人通过牛获得的 1 kg
    需要牛：1 ÷ 20% = 5 kg
    需要玉米：5 ÷ 20% = 25 kg

(3) 人直接吃玉米的 1 kg
    需要玉米：1 ÷ 20% = 5 kg

总需要玉米 = 25 + 25 + 5 = 55 kg
```

> 🔴 **对比：** 如果人全部直接吃玉米（而不是通过鸡和牛），增重 3 kg 只需要 3 ÷ 20% = 15 kg 玉米。食物链越短，能量损失越少——这是为什么提倡"多吃谷物少吃肉"的能量学原理。

## 实验补充：设计制作生态缸

> 高考实验设计题偶尔出现

### 设计要求

| 要求 | 原因 |
|------|------|
| 必须**密封** | 防止外界物质和生物干扰 |
| 放在**散射光**下 | 既保证光合作用，又避免直射导致水温过高 |
| 生物种类适当 | 保证有生产者、消费者、分解者，构成完整生态系统 |
| 动物数量和个体不宜太大 | 防止氧气消耗过快 |

### 评价指标

观察生态缸中生物的存活时间——时间越长，生态缸的稳定性越高。

## 考前速记卡片

| 核心概念 | 一句话 |
|----------|--------|
| 同化量 | = 摄入量 − 粪便量 |
| 传递效率 | = 下一级同化量 ÷ 上一级同化量，一般 10%~20% |
| 能量金字塔 | **唯一不可倒置**的金字塔 |
| $K$ 值 | 环境容纳量，会随环境改变而改变 |
| $K/2$ | 增长速率最快，渔业捕捞后应维持的水平 |
| 碳循环 | 进入群落靠光合作用，出群落靠呼吸作用 |
| 碳中和 | 碳排放 = 碳吸收 |
| 食物链起点 | 一定是生产者（无机环境不写在食物链中） |
| 抵抗力稳定性 | 越复杂的生态系统越高 |
| 恢复力稳定性 | 一般越简单的生态系统越高（但不是绝对的） |
