## 模块一：基因工程四步解题模板 ★★★★★

### 1.1 核心框架速记

```
第1步 → 目的基因的获取
第2步 → 基因表达载体的构建（核心！）
第3步 → 将目的基因导入受体细胞
第4步 → 目的基因的检测与鉴定
```

> 🔑 **口诀：** "获取→构建→导入→检测"，四字四步，绝不混淆。

---

### 1.2 第1步：目的基因的获取

**三种获取方法对比：**

| 方法 | 前提条件 | 特点 |
|------|----------|------|
| **基因组文库筛选** | 不知道基因序列 | 含全部基因（包括内含子），耗时 |
| **cDNA文库筛选** | 只要知道蛋白质能高表达 | 无内含子，在原核细胞中表达时**必须选cDNA** |
| **PCR扩增** | 已知目的基因部分/全部序列 | 快速大量扩增，**需要引物** |
| **人工合成** | 已知完整核苷酸序列 | 适合小基因，成本高 |

> 🔴 **易错：** 原核细胞（如大肠杆菌）**没有RNA剪接酶**，不能加工含内含子的前体mRNA → 所以用原核细胞生产真核蛋白时，必须用 **cDNA**（无内含子）！

**PCR技术要点（高频考）：**

```
PCR三步循环：
① 变性（90~95℃）   ← 氢键断裂，DNA解链为单链
② 退火（55~60℃）   ← 引物与单链互补结合
③ 延伸（70~75℃）   ← Taq酶从引物3'端延伸合成新链
```

| 要素 | 说明 |
|------|------|
| **引物** | 两种，分别与目的基因两端互补，提供延伸起点 |
| **Taq酶** | 耐高温DNA聚合酶，从嗜热水生栖热菌中提取 |
| **dNTP** | 四种脱氧核苷酸，DNA合成的原料 |
| **模板** | 含有目的基因的DNA |

> 🔴 **易错：** PCR只能合成DNA，不能直接合成蛋白质；扩增n轮得2ⁿ个分子，其中只有2个含有原来的链。

---

### 1.3 第2步：基因表达载体的构建 ★★★★

**表达载体必须含有的5个元件：**

```
┌─────────────────────────────────────────────────────┐
│  复制原点 │ 启动子 │ 目的基因 │ 终止子 │ 标记基因  │
└─────────────────────────────────────────────────────┘
```

| 元件 | 位置 | 作用 |
|------|------|------|
| **复制原点** | 载体上 | 使重组质粒在受体细胞内自主复制 |
| **启动子** | 目的基因5'上游 | RNA聚合酶识别与结合的位点，启动转录 |
| **终止子** | 目的基因3'下游 | 发出终止转录的信号 |
| **标记基因** | 载体上 | 筛选导入了重组质粒的受体细胞（如抗性基因） |

**构建步骤（答题格式）：**

```
① 用同种限制酶分别切割目的基因和质粒载体
   → 目的基因产生黏性末端（与质粒切口互补）
② 用DNA连接酶连接目的基因与载体
   → 形成重组质粒（重组DNA分子）
③ 构建完成的表达载体含有：复制原点 + 启动子 + 目的基因 + 终止子 + 标记基因
```

> 🔴 **3个高频易错点：**
> 1. **限制酶切割的是磷酸二酯键**（不是氢键）
> 2. **DNA连接酶连接的也是磷酸二酯键**（不是氢键）
> 3. **启动子≠启动基因**：启动子是DNA上的一段序列，不是基因！

**两种DNA连接酶区别：**

| 酶 | 能否连接平末端 | 能否连接黏性末端 |
|----|:---:|:---:|
| E.coli DNA连接酶 | ❌ | ✅ |
| T₄ DNA连接酶 | ✅ | ✅ |

---

### 1.4 第3步：将目的基因导入受体细胞

| 受体细胞类型 | 导入方法 | 机制 |
|-------------|----------|------|
| **植物细胞**（双子叶植物） | 农杆菌转化法 ★ | Ti质粒上的T-DNA整合进植物细胞染色体 |
| **植物细胞**（单子叶植物、种子） | 基因枪法 / 花粉管通道法 | 物理穿透细胞壁 |
| **动物细胞** | **显微注射法** ★ | 直接注射到受精卵**原核**中 |
| **微生物（原核）** | Ca²⁺处理（感受态细胞法） | 增大细胞膜通透性 |

> 🔴 **农杆菌转化法三要素：**
> - 农杆菌含有 **Ti质粒**（目的基因插入Ti质粒的T-DNA中）
> - T-DNA可整合到**宿主细胞的核DNA**中（稳定遗传）
> - 农杆菌对**双子叶植物和裸子植物**有较强侵染能力，对单子叶植物（水稻、小麦）效果差

---

### 1.5 第4步：目的基因的检测与鉴定

| 检测层次 | 方法 | 结论 |
|----------|------|------|
| **分子水平（DNA）** | DNA分子杂交（Southern印迹） | 目的基因是否整合进染色体 |
| **分子水平（RNA）** | 分子杂交（Northern印迹） | 目的基因是否正常转录 |
| **分子水平（蛋白质）** | 抗原—抗体杂交 | 目的基因是否正常翻译 |
| **个体水平** | 功能性实验 | **最终鉴定**（如抗虫接种/胰岛素注射） |

> 🔑 **答题顺序：** 分子水平（DNA→RNA→蛋白质）→ 个体水平（最终鉴定），不能颠倒！

---

### 1.6 完整解答题模板

**题型一：利用基因工程生产某种蛋白质药物**

```
① 目的基因获取：
   从cDNA文库中获取（在原核受体中表达时必须用cDNA，无内含子）；
   或用PCR技术扩增：提供引物、Taq酶、四种dNTP和含目的基因的模板DNA

② 构建表达载体：
   用同一种限制酶切割目的基因和质粒，产生相同的黏性末端；
   用DNA连接酶连接，形成重组质粒；
   表达载体含有：启动子、目的基因、终止子、标记基因、复制原点

③ 导入受体细胞：
   以大肠杆菌为受体：用Ca²⁺处理制成感受态细胞，再将重组质粒导入

④ 检测与鉴定：
   分子水平：用含目的基因互补序列的探针进行分子杂交（DNA→RNA→蛋白质）；
   个体水平：进行功能性实验（最终鉴定）
```

**题型二：利用农杆菌转化法培育抗病转基因植物**

```
① 将抗病基因插入Ti质粒的T-DNA中，构建重组Ti质粒

② 将重组Ti质粒导入农杆菌；用农杆菌侵染植物叶片/茎段（外植体）

③ T-DNA整合进植物细胞的核DNA中，抗病基因随之稳定表达

④ 通过植物组织培养（脱分化→再分化）获得完整转基因植株

⑤ 检测：让转基因植株接触病原菌，观察是否表现出抗病性（个体水平最终鉴定）
```

## 模块二：细胞工程解题模板 ★★★★

### 2.1 单克隆抗体制备（必考！）

**完整流程：**

```
① 向小鼠体内注射特定抗原
         ↓
② 从小鼠脾脏中获取免疫B淋巴细胞
         ↓
③ 将B淋巴细胞与骨髓瘤细胞在灭活病毒/PEG诱导下融合
         ↓
④ 用特定选择培养基培养（HAT培养基）
   → 未融合的B细胞：死亡（不能无限增殖）
   → 未融合的骨髓瘤细胞：死亡（缺乏HGPRT/TK酶）
   → 同种细胞融合体（B+B / 瘤+瘤）：死亡/增殖有限
   → 杂交瘤细胞（B+瘤）：存活 ✅
         ↓
⑤ 克隆化培养 + 抗体检测（ELISA法）
         ↓
⑥ 筛选出能分泌目标抗体的杂交瘤细胞
         ↓
⑦ 体内（小鼠腹腔）或体外（发酵罐）大量培养
         ↓
⑧ 收集腹水/培养液 → 纯化 → 单克隆抗体
```

**口诀：** "**免疫→融合→选择→克隆检测→生产→纯化**"

**杂交瘤细胞特性（必须能答出）：**

| 特性 | 来源 |
|------|------|
| 无限增殖 | 骨髓瘤细胞（肿瘤性） |
| 产生特异性抗体 | B淋巴细胞（免疫性） |

> 🔴 **4个常考易错点：**
> 1. 用于融合的B淋巴细胞来自**脾脏**（不是胸腺！）
> 2. 融合方法：**灭活病毒**（化学/物理法也可，但灭活病毒最常用）
> 3. 单克隆抗体的优点：**特异性强、灵敏度高、可以大量制备**（三点缺一不可！）
> 4. 同一批杂交瘤细胞产生的抗体针对**同一个抗原决定簇**

---

### 2.2 植物组织培养流程

```
外植体（叶片/茎段/根尖）
    ↓ 消毒灭菌（70%酒精 + 次氯酸钠）
脱分化（加高NAA低BA的培养基）
    ↓
愈伤组织（无定形细胞团，脱离分化状态）
    ↓
再分化（加高BA低NAA的培养基）
    ↓
胚状体或丛芽
    ↓
完整植株（移植到土壤）
```

**激素比例口诀：**
- "**高生（NAA）促根低促芽；高分裂（BA）促芽低促根**"
- 愈伤组织诱导：低BA : 高NAA → 脱分化
- 生芽诱导：高BA : 低NAA → 再分化出芽
- 生根诱导：低BA（或0）: 高NAA → 生根

> 🔴 **原理：** 细胞全能性——已分化细胞的细胞核含有发育所需的全套遗传信息

---

### 2.3 动物细胞培养要点

| 关键点 | 说明 |
|--------|------|
| 分散方式 | **胰蛋白酶**处理（消化细胞间连接，避免接触抑制） |
| 培养基成分 | 葡萄糖 + 氨基酸 + 无机盐 + **动物血清**（提供生长因子） |
| 温度/pH | 36.5±0.5℃ / pH 7.2-7.4 |
| CO₂作用 | **维持培养液pH**（不是提供碳源！） |
| 传代 | 贴壁生长→长满→胰蛋白酶消化→传代 |

## 模块三：限制酶计算专题 ★★★

### 3.1 识别序列与切割产物

**规律：** 限制酶识别的序列通常是**回文序列**（正向反向互补相同）。

**例：EcoRI识别 5'-GAATTC-3'，切割位点在G↓AATTC**

```
切割前：
5'—...G A A T T C...—3'
3'—...C T T A A G...—5'

切割后（产生黏性末端）：
5'—...G           A A T T C...—3'
3'—...C T T A A           G...—5'
         ↑黏性末端(单链)↑
```

### 3.2 常见计算类型

**类型一：计算一个DNA分子被限制酶切成几个片段**

| 情况 | 公式 |
|------|------|
| 线性DNA | 切割n次 → n+1个片段 |
| 环状DNA（质粒） | 切割n次 → n个片段 |

> **例：** 一个环状质粒有3个EcoRI切割位点 → 切割后产生 **3个** 片段

**类型二：分析电泳结果，判断目的基因是否导入**

```
原质粒（对照）：EcoRI切割 → 2个条带（如2kb和4kb）
重组质粒：EcoRI切割 → 3个条带（多出目的基因大小的条带）

→ 若电泳结果条带比原质粒多一条，且大小符合目的基因，
  则说明目的基因成功导入！
```

**类型三：用两种限制酶（双酶切）防止自连**

> 用两种不同限制酶切割，使目的基因两端产生**不同的黏性末端**，
> 只能定向插入载体，不能自身环化自连 → 提高重组效率

## 模块四：发酵工程综合题答题套路 ★★

### 4.1 发酵工程流程（答题顺序）

```
① 菌种选育（自然选育/诱变育种/基因工程育种）
② 配制培养基（碳源 + 氮源 + 水 + 无机盐 + 特殊营养物质）
③ 灭菌（培养基：高压蒸汽灭菌/灼烧灭菌/干热灭菌）
④ 接种（无菌操作，酒精灯旁）
⑤ 发酵（控制温度、pH、通气量、搅拌速度）
⑥ 产品分离提纯（过滤、离心、蒸馏、沉淀等）
```

### 4.2 三种灭菌方法对比（常考）

| 灭菌方法 | 温度/条件 | 适用对象 | 注意事项 |
|----------|-----------|----------|----------|
| **高压蒸汽灭菌**（灭菌锅） | 121℃，100kPa，15-30min | 培养基、无菌水、玻璃器皿 | **先调pH，再灭菌**！ |
| **干热灭菌** | 160-170℃，1-2h | 金属器具、玻璃器皿 | 不能用于培养基（会破坏营养成分） |
| **灼烧灭菌** | 酒精灯火焰直接灼烧 | 接种环、试管口、镊子 | 灼烧后要冷却再使用 |

> 🔴 **培养基pH调节时机：** 先调节pH → 再高压蒸汽灭菌（灭菌可能改变pH，所以先调，不能反过来）

### 4.3 微生物计数方法

| 方法 | 计数范围 | 误差方向 | 原因 |
|------|----------|----------|------|
| **稀释涂布平板法** | 30-300菌落/平板 | 偏低 | 两菌相邻无法区分，计为1个菌落 |
| **血细胞计数板直接计数** | — | 偏高 | 死菌也被计入 |

**稀释涂布平板法计算公式：**
```
每克样品菌落数 = 菌落数 ÷ 涂布体积（mL）× 稀释倍数
```

> 例：在10⁻⁶稀释液中，涂布0.1mL，平板上长出150个菌落
> → 每克样品中含菌量 = 150 ÷ 0.1 × 10⁶ = **1.5 × 10⁹个/g**

## 模块五：高考综合大题实战演练

### 5.1 基因工程综合大题（仿真题）

**题目：**
科学家将人的胰岛素基因导入大肠杆菌，使其大量生产胰岛素。请回答：

（1）获取人胰岛素基因时，应选用 _____ 文库（填"基因组"或"cDNA"），原因是 _____。

（2）将胰岛素基因与质粒连接时，所用工具酶是 _____（两种）。切割后，质粒和胰岛素基因产生 _____ 末端，再用DNA连接酶连接，形成 _____。

（3）将重组质粒导入大肠杆菌，需先用 _____ 处理大肠杆菌，使其成为感受态细胞。

（4）检测大肠杆菌是否成功合成胰岛素蛋白，最终需要在 _____ 水平进行鉴定，方法是 _____。

**参考答案：**
1. **cDNA**文库；大肠杆菌（原核生物）没有RNA剪接酶，不能加工含有内含子的mRNA前体，所以必须用无内含子的cDNA
2. **限制酶（限制性核酸内切酶）** 和 **DNA连接酶**；**相同的黏性**；**重组质粒（重组DNA分子）**
3. **Ca²⁺**（氯化钙溶液）
4. **个体**水平；将含胰岛素蛋白的提取物注射到患糖尿病的小鼠体内，观察其血糖是否恢复正常（或：用抗原-抗体杂交法检测蛋白质水平 → 但个体水平是最终鉴定）

### 5.2 单克隆抗体综合题（仿真题）

**题目：**
请简述用杂交瘤技术制备甲型流感病毒单克隆抗体的过程，并说明单克隆抗体的优点。

**参考答案：**

**制备过程：**
① 将灭活的甲型流感病毒注射到小鼠体内，进行免疫；
② 从小鼠脾脏中获取免疫B淋巴细胞；
③ 在**灭活病毒（或PEG）** 诱导下，将B淋巴细胞与骨髓瘤细胞融合，获得杂交细胞；
④ 用 **HAT选择培养基** 筛选，只有杂交瘤细胞能存活；
⑤ 进行**克隆化培养**，同时用**ELISA（或抗原-抗体杂交）** 检测各克隆产生的抗体；
⑥ 筛选出能分泌针对流感病毒抗体的杂交瘤细胞；
⑦ 注射到小鼠腹腔（或体外培养）→ 大量获得单克隆抗体；
⑧ 提取并纯化单克隆抗体。

**单克隆抗体的优点（三点必须全写）：**
- **特异性强**（只与一种抗原决定簇结合）
- **灵敏度高**（检测浓度低的抗原）
- **可以大量制备**（杂交瘤细胞能无限增殖）

---

## 综合对比：三大技术的"全能性"基础

| 技术 | 使用的全能性 | 全能性来源 |
|------|-------------|-----------|
| 植物组织培养 | 植物细胞全能性 | 高度分化的细胞核仍含全套遗传信息 |
| 克隆动物（体细胞核移植） | 动物细胞核全能性 | 体细胞核 + 去核卵母细胞胞质 |
| 胚胎干细胞（ES细胞）培养 | ES细胞全能性 | 来源于囊胚内细胞团，未分化 |

> 📌 **注意区分：**
> - **植物细胞**：细胞本身具有全能性（全能性最高）
> - **动物体细胞**：细胞本身全能性受限，但**细胞核**仍有全能性（多莉羊的证据）
> - **受精卵**：全能性最高的动物细胞

### 6.1 转基因生物的安全性争议

#### 支持转基因技术的论点（正确观点）

| 观点 | 依据 |
|------|------|
| 可提高农作物产量、品质和抗性 | 抗虫/抗病/抗旱转基因作物已大规模推广 |
| 基因工程是基因水平的定向改造 | 比传统育种更精确，不会随机引入大量外源基因 |
| 目前上市的转基因食品已经过严格安全评估 | 批准上市前需经过毒理学、过敏性等多项检测 |
| 基因重组本来就是自然界普遍存在的现象 | 细菌质粒传递、病毒整合宿主DNA等均属基因重组 |

#### 对转基因生物持谨慎态度的论点（需要关注的风险）

| 担忧 | 说明 |
|------|------|
| **基因污染** | 转基因植物的花粉可能与野生近缘种杂交，将抗性基因扩散到野外（破坏生态平衡） |
| **食品安全** | 外源蛋白可能引起部分人群过敏；长期效应有待研究 |
| **生物多样性** | 超级杂草/超级害虫的出现可能威胁生态平衡 |
| **伦理争议** | 对生命进行人工改造，涉及"造物主"问题的伦理讨论 |

> 🔴 **高考判断易错：**
> - ❌ "转基因食品含有外源DNA，吃了会改变人的遗传物质" → **错误！** 食物中的DNA会被消化道分解为核苷酸，不会直接整合进人体基因组
> - ❌ "转基因技术是将基因任意改造，是不安全的" → **错误！** 基因工程是定向、精确的基因操作
> - ✅ "转基因生物可能对生态环境产生基因污染" → **正确**

---

### 6.2 克隆技术的伦理争议

#### 什么是克隆？

```
体细胞核移植（SCNT）：
供体体细胞的细胞核  →  去核卵母细胞
         ↓ 融合（灭活病毒/电刺激）
     重组细胞（含供体遗传信息）
         ↓ 体外培养、发育
     胚胎  →  代孕母体  →  克隆动物
```

> **关键点：** 克隆动物的**核遗传物质**与供体相同，但**细胞质遗传物质（少量线粒体DNA）** 来自卵母细胞供体 → 所以克隆动物**不完全等同于**供体

#### 各国对克隆技术的态度

| 技术类型 | 多数国家立场 | 理由 |
|----------|-------------|------|
| **治疗性克隆**（克隆胚胎，取干细胞，不发育成人） | ✅ 部分允许 | 可用于治疗疾病（帕金森、糖尿病等），不涉及独立个体 |
| **生殖性克隆**（克隆人） | ❌ 明确禁止 | 违背人类尊严、存在严重伦理问题 |

> 🔴 **高考判断标准：**
> - ✅ "克隆有助于保护濒危物种" → **正确**（辅助繁殖技术）
> - ✅ "克隆技术可用于器官移植的研究" → **正确**（治疗性用途）
> - ❌ "支持克隆人，因为可以延续生命" → **错误**（违反伦理，各国明令禁止）
> - ❌ "克隆人和被克隆者完全相同" → **错误**（后天环境/细胞质遗传物质不同，不完全相同）

---

### 6.3 生物武器的伦理问题

#### 什么是生物武器？

利用**病原微生物（细菌、病毒、毒素）** 作为战争手段，造成大规模伤亡和破坏。

| 特点 | 说明 |
|------|------|
| **危害性极大** | 一旦释放难以控制，可造成大规模传染病爆发 |
| **无差别伤害** | 不能区分战斗人员和平民 |
| **长期影响** | 某些病原体可在环境中长期存活，影响生态系统 |
| **违反国际法** | 《禁止生物武器公约》（1975年生效）：明确禁止开发、生产、储存生物武器 |

> 🔴 **高考选择题判断套路（全国乙卷风格）：**
> - 凡涉及"禁止生物武器"的选项 → 通常为**正确选项**（符合国际公约立场）
> - 凡说"生物武器危害可控/可定向使用" → **错误**

---

### 6.4 生物技术伦理判断速查表

| 问题类型 | 正确态度（高考官方立场） |
|----------|------------------------|
| 转基因食品 | **谨慎推广，需严格检测评估**；已批准上市的是安全的 |
| 转基因作物商业化 | **利大于弊，但要建立安全评价体系** |
| 克隆人 | **坚决反对**（违背伦理，各国明令禁止） |
| 治疗性克隆 | **有条件支持**（用于医学研究，不发育成完整个体） |
| 生物武器 | **坚决反对**（违反国际公约，危害全人类） |
| 基因歧视 | **坚决反对**（不得因基因差异歧视个体） |
| 人类基因组计划 | **支持**（揭示人类遗传信息，推动医学发展）；但个人基因隐私需保护 |
